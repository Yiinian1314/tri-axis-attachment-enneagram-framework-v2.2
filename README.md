## 二、使用场景

### 场景 1: 咨询师个人学习(初学者)
1. 先 读 `01-理论架构与咨询指南.md`, 把握整体框架
2. 翻 `02-咨询师速查工具卡.md`, 现场查阅
3. 根据 你接待的来访者构型, 读对应的 `05-构型深度展开/构型XX.md`
4. 听/读 `06-咨询录音脚本/`, 学习咨询话术与节奏
5. 做 `07-咨询师自我校准/咨询师自我校准-自评工具.md`, 了解自己的"咨询师构型"
6. 结合 `09-案例库/`, 看真实案例的分析与教学要点

### 场景 2: 机构内训
- 直接 用 `03-培训PPT/attachment-enneagram-consultation-training.pptx` 进行 2.5 小时培训
- 培训后分发 `01-理论主文档` 给学员
- 学员可按需查阅 `05-构型深度展开`、`06-咨询录音脚本`、`09-案例库`
- 进阶培训用 `07-咨询师自我校准` 和 `08-督导专题包`

### 场景 3: 实际咨询中
- 评估阶段: 用 `02-速查工具卡` 快速定位来访者构型
- 咨询中: 用 `05-构型深度展开` 中对应构型的"咨询路线图"
- 咨询后: 总结时参考 `06-咨询录音脚本` 的结构
- 咨询师自身: 用 `07-咨询师自我校准-反移情手册` 识别自己的反移情
- 督导时: 用 `08-督导专题包-12个构型的督导要点` 作为督导参考

### 场景 4: 给来访者
- 评估后: 给来访者 `04-来访者自助手册.pdf`
- 让来访者作为家庭作业, 持续做手册里的练习

### 场景 5: 督导
- 督导前: 咨询师用 `08-督导专题包-督导流程与记录模板.md` 准备案例
- 督导中: 用 `08-督导专题包-12个构型的督导要点` 作为督导参考
- 危机情境: 用 `08-督导专题包-危机处理与转介指南.md` 处理危机
- 转介时: 用 `08-督导专题包-危机处理与转介指南.md` 的转介流程

### 场景 6: 培训学员 / 教学
- 示范: 用 `09-案例库/案例01-05` 展示"完整咨询过程"
- 分析: 用 `09-案例库/案例分析框架与教学要点模板.md` 的双轴框架
- 讨论: 围绕案例的关键临床决策和反移情事件

## 三、核心方法论

### 1. 三轴诊断模型(v2.0 重大升级)
焦虑维度 × 回避维度 × 九型主型 × 客体关系维度 = 来访者的"位置"

| 轴 | 关注 | 临床抓手 |
|---|------|---------|
| 依恋 | 怎么做 | 身体调节、关系策略 |
| 九型 | 为什么 | 核心恐惧、防御、整合方向 |
| 客体关系 | 内在世界长什么样 | 投射性认同、内在图、温尼科特时刻 |

### 2. 三轨成长模型(v2.0 重大升级)
```
轨道 A(依恋修复) → 轨道 B(九型整合) → 轨道 C(客体关系修补)
```

- 轨道 A:稳定身体与关系
- 轨道 B:深挖动力
- 轨道 C:修补内在图

### 3. 三大核心原则
1. 依恋告诉你来访者"在" 关系中 "怎么反应" (How)
2. 九型告诉你来访者"为什么 "这样反应(Why)
3. 客体关系告诉你来访者的"内在世界长什么样" (What shape)— v2.0 新增
4. 咨询师的工作不是"修理" 来访者, 是陪他/她重新认识自己, 然后主动选择

### 4. 顺序铁律
触碰核心恐惧前, 先确保安全基地够稳。 修补内在图前, 先稳定动力。 顺序颠倒, 会复制早年创伤而非修复。

### 5. 累积原则
改变, 来自"重复" 而不是"刺激"。 一次咨询不解决问题, 多次"小实验" 的累积, 才是修复的实质。

### 6. 咨询师原则(本套件特别强调)
- 咨询师的"未完成议题" 是核心反移情来源
- 不同的来访者构型, 激活咨询师不同的"未完成议题"
- 咨询师必须有自己的个人咨询 + 督导

### 7. 督导原则(本套件特别强调)
- 督导不是"通用" 的, 是"构型定制" 的
- 不同构型来访者, 督导的"重点" 不同
- 危机处理有标准化流程

### 8. 客体关系原则(v2.0 新增)
- 关系疗愈不来自"不破裂", 而来自"破裂后的修复"
- "足够好" 的咨询师 > "完美" 的咨询师
- 投射性认同是"双向" 的, 不是来访者单方面的
- 内在图的修补需要"新经验", 不是"认知教育"

## 五、本轮新增模块的使用指南

### 模块 7: 咨询师自我校准

目标用户: 所有情感咨询师

核心价值: 反移情识别、耗竭管理、自我成长

使用方式:
1. 新咨询师: 先做 `07-咨询师自我校准-自评工具.md` 的自评, 了解自己的"咨询师构型"
2. 所有咨询师: 每季度做一次自评 + 自评结果与督导讨论
3. 咨询中遇到反移情时: 翻 `07-咨询师自我校准-反移情手册.md`, 找对应构型的反移情
4. 感觉耗竭时: 翻 `07-咨询师自我校准-耗竭识别与恢复.md`

### 模块 8: 督导专题包

目标用户: 做督导的咨询师 + 被督导的咨询师

核心价值: 构型定制督导、危机处理、转介

使用方式:
1. 做督导的咨询师: 用 `08-督导专题包-督导流程与记录模板.md` 标准化督导
2. 被督导的咨询师: 用 `08-督导专题包-12个构型的督导要点.md` 自查
3. 遇到危机时: 立即用 `08-督导专题包-危机处理与转介指南.md`
4. 考虑转介时: 用 `08-督导专题包-危机处理与转介指南.md` 的转介流程

### 模块 9: 案例库

目标用户: 咨询师培训学员、做督导的咨询师、新咨询师

核心价值: 真实脱敏案例、双轴分析框架、教学要点

## 九、危机处理速查

任何危机情境, 立即用 `08-督导专题包-危机处理与转介指南.md`

### 危机识别

- 来访者: 自杀风险 / 他杀风险 / 急性精神状况 / 急性创伤事件
- 咨询师: 严重耗竭 / 精神问题 / 物质滥用 / 伦理违规
- 机构: 大规模事件 / 媒体关注 / 法律诉讼

### 危机处理 5 步

1. 安全第一: 立即评估安全风险
2. 不单打独斗: 立即联系督导、精神科、危机热线
3. 不轻看: 所有自杀/他杀念头都认真对待
4. 不诊断: 危机处理是评估 + 处理, 不是诊断
5. 事后复盘: 危机后必须复盘(个人 + 督导 + 必要时机)

### 危机热线

- 110 (警察)
- 120 (急救)
- 119 (消防)
- 当地心理危机干预中心
- 全国心理援助热线

- 依恋理论 (Attachment Theory): John Bowlby, Mary Ainsworth
- 九型人格 (Enneagram): Ichazo, Naranjo, Riso-Hudson
- 客体关系理论 (Object Relations Theory): Melanie Klein, Donald Winnicott, Wilfred Bion, Otto Kernberg, Heinz Kohut
- 情绪聚焦治疗 (EFT): Sue Johnson, Les Greenberg
- 图式治疗 (Schema Therapy): Jeffrey Young
- 复杂性创伤治疗 (Complex Trauma): Judith Herman, Bessel van der Kolk

本套件的核心创新在于:
1. 三轴诊断模型 (v2.0 升级): 依恋 × 九型 × 客体关系的三轴交叉诊断
2. 三轨成长模型 (v2.0 升级): 依恋修复 → 九型整合 → 客体关系修补
3. 三轴融合框架 (v2.2 升级): 主文档 1.3-1.5(层级关系 / 对话机制 / 冲突优先级 / 12 构型速览图谱)
4. 构型定制咨询师反移情手册: 12 个构型的咨询师反移情
5. 构型定制督导要点: 12 个构型的督导关注点
6. 12 构型内在图谱 (v2.0 升级): 12 个构型的自体表征 / 客体表征 / 关系模板
7. 温尼科特时刻与破裂修复 (v2.0 升级): 5 个温尼科特时刻 + 4 步破裂-修复
8. 5 阶段咨询流程 + 整合性干预
