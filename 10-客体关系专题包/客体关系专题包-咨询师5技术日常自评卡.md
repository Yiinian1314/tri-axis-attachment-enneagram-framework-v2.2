## 二、技术 1:投射性识别(5 步法)

### 2.1 核心问题

> "我的情绪是我的,还是来访者的?"

### 2.2 5 步法速查

| 步 | 做什么 | 时间 |
|---|-------|------|
| 1. 停 | 觉察"我有强烈情绪" | 1 秒 |
| 2. 命名 | "这情绪真的是我的吗?" | 5 秒 |
| 3. 追溯 | "来访者做了什么触发的?" | 10 秒 |
| 4. 反推 | "如果这是投射,来访者内在的什么被激活?" | 15 秒 |
| 5. 选择 | 不进入投射;也不强硬拒绝;命名投射本身 | 30 秒 |

## 三、技术 2:温尼科特时刻

### 3.1 核心问题

> "当来访者展示真实的脆弱,我怎么'足够好'地回应?"

### 3.2 5 个关键

| 关键 | 描述 |
|------|------|
| 沉默 > 长篇共情 | 大多数情况下,沉默比说话更"接住" |
| 简单 > 复杂 | "谢谢你告诉我" > 长篇分析 |
| 在场 > 共情 | "在" 比"共情"更重要 |
| 不评判 > 评判 | "你的想法是 OK 的" > "你应该..." |
| 不修复 > 修复 | 看见来访者 > 修复来访者 |

### 3.3 日常自评卡

咨询日期:________  来访者(化名):________

| 问题 | 0 | 1 | 2 | 3 |
|------|---|---|---|---|
| 1. 我是否注意到来访者展示了"真"(脆弱/未说的话)? | □ | □ | □ | □ |
| 2. 我是否没有填满沉默(沉默 ≥ 3 秒)? | □ | □ | □ | □ |
| 3. 我的回应是否简单(< 3 句)? | □ | □ | □ | □ |
| 4. 我是否不评判来访者的"真"? | □ | □ | □ | □ |
| 5. 我是否不试图"修复"来访者的脆弱? | □ | □ | □ | □ |
| 6. 我是否在场(眼神/身体朝向)? | □ | □ | □ | □ |

总分:___/18

### 3.4 5 个"禁忌"自查

- [ ] 我没有"立即分析"来访者的脆弱
- [ ] 我没有"立即自我暴露"("我也经历过")
- [ ] 我没有"立即修复"来访者的感受
- [ ] 我没有"立即填满"沉默

## 四、技术 3:破裂-修复循环

### 4.1 核心问题

> "咨询师失误了,我怎么'真诚地修复?"

### 4.2 4 步流程

| 步 | 做什么 | 话术示例 |
|---|-------|----------|
| 1. 认错 | "这是我做错了"(不借口/不解释/不辩护) | "对不起,我迟到了。这是我的错。" |
| 2. 命名影响 | "我想这可能让你感到..." | "我想这可能让你感到被忽视。" |
| 3. 连接 | "但我想让你知道,我仍然在这里" | "但我仍然在这里,我关心你。" |
| 4. 询问 | "你能告诉我,你的感受吗?" | "你愿意说说你的感受吗?" |

## 五、技术 4:命名分裂

### 5.1 核心问题

> "当来访者'全好/全坏'快速切换,我怎么示范'整合'?"

### 5.2 识别"分裂"的 5 个信号

- [ ] 来访者反复说"咨询师是世界上最好/最坏的人"
- [ ] 来访者反复在"上帝"和"魔鬼"之间切换
- [ ] 来访者反复"我今天状态超好"(明天"我又跌入谷底")
- [ ] 来访者反复把伴侣/家人/同事"理想化"或"贬低"
- [ ] 来访者反复用"全好"或"全坏"形容自己

### 5.3 回应模板

> "我注意到,你说我'最懂你'时,跟上周说我'不懂你',是两个不同的人。这两个'我'都是你真实的部分——上周的你不相信我,今天的你信任我。这两个'我'都还在,只是今天信任的部分更多一些。"

## 六、技术 5:内在图改写

### 6.1 核心问题

> "当来访者自发叙述新经验,我怎么强化它?"

### 6.2 关键识别:新经验的"5 个信号"

- [ ] 来访者说"我以前会崩,但今天我没有"
- [ ] 来访者说"我以前会说'你爱我吗',今天我没有"
- [ ] 来访者说"我以前会陷入推-拉,今天我没有"
- [ ] 来访者说"我以前会逃避,今天我待了"
- [ ] 来访者自己注意到自己的变化

### 6.3 强化的关键

- 不是咨询师"告诉"来访者"你变了"
- 是咨询师"强化"来访者自己注意到的变化
- 强化话术:"你注意到了吗?你的反应变了。"

## 九、5 技术的"高难度"场景(自评卡附)

### 9.1 高难度 1:咨询师被反复测试

场景:来访者反复问"你爱我吗""你不会走吧""你真的关心我吗?"

应该用:T1 投射性识别 + T2 温尼科特时刻

自评点:
- [ ] 我没有"反复保证"("我保证")
- [ ] 我没有"被卷进"投射
- [ ] 我没有"反复解释"
- [ ] 我不断"接住"来访者的问

### 9.2 高难度 2:咨询师被来访者"卷进"投射

场景:来访者反复说"你也不关心我",然后咨询师真的感觉"不关心"了

应该用:T1 投射性识别 + T3 破裂-修复

自评点:
- [ ] 我觉察了自己被卷进
- [ ] 我没有"跟着"来访者"不关心"
- [ ] 我没有"否认"来访者的"不关心"
- [ ] 我有"承接"——"这个感受我听到了"

### 9.3 高难度 3:咨询师出现严重失误

场景:咨询师忘记了来访者说过的重要事,或说错了严重伤害来访者的话

应该用:T3 破裂-修复(最核心)

自评点:
- [ ] 我承认了错误(不辩护)
- [ ] 我没有"归咎于来访者"
- [ ] 我主动发起修复(不等来访者) 
- [ ] 我给了来访者"说"的空间(不强迫)
### 9.4 高难度 4:来访者陷入"全好/全坏"快速切换

场景:来访者今天说"你是世界上最好的咨询师",明天说"你根本不是真正关心我"

应该用:T4 命名分裂

自评点:
- [ ] 我没有"跟着"来访者"全好"
- [ ] 我没有"跟着"来访者"全坏"
- [ ] 我命名了"两个都是你"
- [ ] 我示范了"整合"(灰色) 

### 9.5 高难度 5:来访者自发叙述"新经验"

场景:来访者说"今天我伴侣迟到了,我没有立即问'你爱我吗',我等了"

应该用:T5 内在图改写

自评点:
- [ ] 我注意了来访者的"新"
- [ ] 我没有"认为理所当然"
- [ ] 我强化了"新经验"("你注意到了吗?")- [ ] 我连接到"旧模式"(让变化被看见)

- [ ] 每 周和督导讨论 1 次

### 10.3 第 5-6 个月:自然应用

- [ ] 每 2-3 次咨询后,选 2-3 个技术做自评- [ ] 不要求"自然",只要求"完整"
- [ ] 周 1 次完整 5 技术自评
- [ ] 每 周和督导讨论 1 次

- [ ] 自评的目标:"我的技术成熟度在哪里"- [ ] 每 月和督导讨论 1 次

## 十一、5 技术的"常见错误"与"如何避免"

### 11.1 T1 投射性识别 — 常见错误

| 错误 | 描述 | 避免 |
|------|------|------|
| 否认反移情 | "我没有反移情" | 承认反移情是正常的 |
| 过度识别 | 把所有情绪都归为投射 | 区分"我的"和"来访者的" |
| 不说 | 知道有反移情,不说 | 说出来(对来访者或对督导) |

### 11.2 T2 温尼科特时刻 — 常见错误

| 错误 | 描述 | 避免 |
|------|------|------|
| 过度共情 | 长篇共情,没沉默 | 默默 + 简单回应 |
| 自我暴露 | "我也经历过" | 不抢镜 |
| 立即修复 | "你不会孤独的" | 不修复;只承认 |

### 11.3 T3 破裂-修复 — 常见错误

| 错误 | 描述 | 避免 |
|------|------|------|
| 否认破裂 | "这不算破裂" | 承认破裂 |
| 辩护 | "我迟到是因为..." | 不辩护;承认错误 |
| 归咎 | "你太敏感了" | 不归咎于来访者 |
| 跳过 | "算了,继续" | 修复;承认错误 |

### 11.4 T4 命名分裂 — 常见错误

| 错误 | 描述 | 避免 |
|------|------|------|
| 跟着"全好" | 接受来访者的"全好" | 不跟着;示范"整合" |
| 跟着"全坏" | 接受来访者的"全坏" | 不跟着;示范"整合" |
| 试图修复 | "你不应该这样想" | 不修复;示范"灰色" |

### 11.5 T5 内在图改写 — 常见错误

| 错误 | 描述 | 避免 |
|------|------|------|
| 不强化 | 来访者说"我今天没崩",咨询师没反应 | 强化;这是核心时刻 |
| 告诉来访者"你变了" | "你和以前不一样了"(咨询师说) | 让来访者自己说"我变了" |
| 连接受访者"以后还会崩" | "你以后可能还会崩" | 承认过去;强化现在 |

## 十三、最后的提醒

> 5个技术不是"学会"就好了 — 它们是"做到"的事。
>
> 新咨询师的成长路径:不知道 → 知道做不到 → 想起生硬 → 自然不完美 → 自然完整。
>
> 每个阶段都有价值:不知道不是坏的;知道做不到也不是坏的;只要你在成长。
>
> 愿你:在每次咨询后花5分钟自评;在每次督导中真诚地讨论;在每个来访者身上慢慢学会这5个技术。
>
> 愿你:在自己的未完成议题上继续工作;在自己的反移情中慢慢成长;在自己的修复中慢慢接受自己。
