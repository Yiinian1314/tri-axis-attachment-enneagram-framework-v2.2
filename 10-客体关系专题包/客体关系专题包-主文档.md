> 本次新增模块
> 理论补充:从双轴(依恋 × 九型)升级到三轴(依恋 × 九型 × 客体关系)
> 适用对象:所有使用本套件的情感咨询师

## 一、为什么本模块是"必需"的

### 1.1 前两轮套件的"盲点"

在前两轮的套件里,我们使用了双轴诊断:
- 依恋(行为)
- 九型(动力)

这能回答两个问题:
- 来访者在关系中怎么做?(依恋)
- 来访者为什么这样做?(九型)

但回答不了第三个问题:
- 来访者内在的世界长什么样?

### 1.2 一个临床案例

> 客户 L,女,32 岁,焦虑型依恋 + 2 号助人型。
>
> 来访者 5 次咨询后,主诉仍然:
> "我知道我付出太多了——我老公还是不珍惜我。"
> "我知道我应该减少付出——但我做不到。"
> "我知道这不是他的问题——但我看到他在玩手机我就会生气。"

双轴分析(原套件能做的):
- 焦虑型:她害怕被抛弃,所以反复"追"
- 2 号:她用"被需要"换"被爱"

为什么"知道但做不到"?

这是双轴看不到的:
- 来访者的内在世界里,有一个"客体表征"——"他人是不会主动爱我的"
- 来访者的自体表征——"我必须有用才值得存在"
- 关系模板——"我付出=他爱"
- 投射性认同——她把"无助"投射到丈夫身上,丈夫变得"真的"像无法自理,然后她"验证"了内在图("我必须照顾他")

第三次咨询的目标,应该是"修补内在图",而不是"再教育她认知"。

这是客体关系的贡献。

### 1.3 三轴的整合价值

| 维度 | 回答的问题 | 临床抓手 |
|------|------------|----------|
| 依恋 | 怎么做 | 身体调节、关系策略 |
| 九型 | 为什么 | 核心恐惧、防御、整合方向 |
| 客体关系 | 内在世界长什么样 | 投射性认同、内在图、温尼科特时刻 |

三者缺一不可:
- 没有依恋:看不见当下激活
- 没有九型:看不见动力
- 没有客体关系:看不见"为什么知道但做不到"

## 三、客体关系理论的 5 个核心概念

### 概念 1:内在客体关系(Internal Object Relations)

我们每个人在内心都形成了重要他人的心理表征——"内在客体"。这些表征是配对存在的:

- 自体表征(Self-Representation):"我是什么样的"
- 客体表征(Object-Representation):"他人是什么样的"

例:

| 自体表征 | 客体表征 | 关系模板 |
|----------|----------|----------|
| 我是不重要的 | 他人会忽略我 | 在关系中我会被忽略 |
| 我是有缺陷的 | 他人是完美的 | 我要修补自己才配被爱 |
| 我是会死的 | 他人是会离开的 | 我要抓紧任何人 |

这些表征早年形成,并持续到当下。它们不是事实,而是早年经验的固化。

### 概念 2:投射性认同(Projective Identification)

Klein 提出的核心概念:
> 把自己的某部分(通常是难以接受的)放到另一个人身上,然后通过与对方的互动"验证"这一点。

关键特征:
- 不是"投射"——不是把情绪"扔"到对方身上,然后跑
- 是"诱导"——把自己的某部分"放到"对方身上,然后通过互动让对方真的变成那样

例(一个混乱 4 号的咨询):
- 来访者内在有"我是被抛弃的"
- 来访者把"被抛弃"投射到咨询师身上
- 来访者反复说"你是不是觉得我烦"、"你是不是已经放弃我了"
- 咨询师开始真的感觉"这来访者真烦"
- 咨询师真的在第 6 次咨询时"走神"
- 来访者"看见"了"咨询师真的放弃了我"——内在图被验证

这个过程是"双向"——不只是来访者单方面投射,咨询师共振或激活了这个投射性认同。

### 概念 3:分裂(Splitting)

Klein 描述的"偏执-抑郁位"中的核心特征:
> 把好与坏完全分开——好的客体(全好)与坏的客体(全坏)互不相关。

正常发展路径:
- 婴儿早期:只能容纳"全好妈妈"或"全坏妈妈"
- 健康发展:学会容纳"妈妈有时好有时坏"
- 进一步发展:学会容纳"我有时好有时坏"

未整合的分裂:
- 理想化与贬低的快速切换
- "上周咨询师是世界上最好的人,这周是世界上最坏的"
- "他一开始完美,现在完全是怪物"

临床意义:
- 分裂是未整合的心智特征——来访者还没学会"灰色"
- 咨询师不进入"全好"(被理想化)或"全坏"(被贬低)的角色
- 咨询师示范整合:"上周的你不信我,今天的你信我。这两个都是你"

### 概念 4:温尼科特"足够好的母亲"

Winnicott 的核心洞见:
> 完美不是好,"足够好"才是好。

核心论点:
- 母亲不必永远正确、不必永远在场、不必永远共情
- 母亲只需要"在大多数时候'足够好'"——这就够婴儿建立安全依恋
- 母亲必然会有"不够好"的时候(不可能永远完美)——这不是问题
- 关键是修复——"不够好"之后,母亲"回到"婴儿身边,修复断裂

对咨询的意义:
- 咨询师不必永远完美——失误是必然的
- 失误 + 真诚修复 > 永远完美
- 这是最深层的疗愈

### 概念 5:自体客体(Selfobject)经验

Kohut 提出"自体客体":
> 一种被体验为自己的一部分的人——虽然客观上他是另一个人。

三种关键功能:
- 镜映(mirroring):"我被看到、被肯定"
- 理想化(idealization):"我仰望一个人,从他那里获得力量"
- 孪生/同类感(twinship):"我属于这里,我和他们一样"

自体客体的失败:
| 失败 | 来访者的等式 |
|------|--------------|
| 镜映失败 | "我必须自己变得特别才能被看见"(3 号、4 号) |
| 理想化失败 | "我必须自己变强才能保护自己"(5 号、8 号) |
| 孪生失败 | "我必须消失才不会引起注意"(9 号) |

## 四、三轴诊断的实际操作

### 4.1 升级后的诊断模型

```
                  客体关系维度(内在世界结构)
                          ↑
                          |
                  +-------+-------+
                  |               |
                  |   来访者     |
                  |   内在图     |
                  |               |
                  +-------+-------+
                          |
                          ↓
   +-------------+--------------+--------------+
   |             |              |              |
依恋(行为)  九型(动力)  客体关系(结构)  身体(过程)
怎么做    为什么    内在世界   当下在身体里
```

### 4.2 客体关系维度的临床评估

核心评估问题:

1. "你小时候觉得'我是什么样的'?"(自体表征)
2. "你小时候觉得'他人是什么样的'?"(客体表征)
3. "在关系中,你最怕发生什么?为什么会怕?"(关系模板 + 投射性认同)
4. "你第一次有这种感受,是在什么时候?跟谁?"(自体客体的早期经验)
5. "现在你身边的 X(伴侣/家人/朋友),是否让你'想起'那个最早的'他者'?"(移情对象)

### 4.3 综合诊断书写模板

```

--------------

- 压力方向:__号
- 整合方向:__号

三、客体关系诊断(本次新增)
---------------------------
- 自体表征:"我是什么样的"
- 客体表征:"他人是什么样的"
- 关系模板:__ = __
- 投射性认同模式:投射 __ → 对方 __

- 自体客体需求:镜映 / 理想化 / 孪生
```

## 五、三轨成长模型(本次升级)

### 5.1 为什么"三轨"?

单轨的不足:
- 只做依恋修复:来访者"知道"但"做不到"——动力没被处理
- 只做九型整合:来访者"看见"但"仍被困"——内在图没被修补
- 三轨都做但顺序错:前面工作被推翻

三轨的顺序:
```
轨道 A(依恋修复)
  ↓ 先稳定身体与关系
轨道 B(九型整合)
  ↓ 再深挖动力
轨道 C(客体关系修补)
  ↓ 最后修补内在图
```

### 5.2 三轨的并行

不是严格按顺序。在实际咨询中,三轨是并行进行的:
- 主体在轨道 A
- 在合适时进入轨道 B
- 在有破裂时进入轨道 C

关键:不跳跃——不直接做 C 而跳过 A 和 B。

### 5.3 三轨的目标

| 轨道 | 起点 | 终点 |
|------|------|------|
| A:依恋修复 | 来访者被"激活"(黏附/退缩/推拉) | 来访者能在压力下保持"在场" |
| B:九型整合 | 来访者被"类型自动导航" | 来访者能"主动选择"而非"被模式带着走" |
| C:客体关系修补 | 来访者有"我=__ / 他人=__"的内在图 | 来访者能拥有"新经验"——在关系中被接住、被尊重、被看见 |

## 六、5 个核心干预技术(本次新增)

### 技术 1:投射性认同识别
核心问题:"这股情绪是我的还是你的?"

### 技术 2:温尼科特时刻
核心姿态:"沉默 + 眼神 + 我在这里 = 接住"

### 技术 3:破裂-修复循环
核心流程:"咨询师示弱 + 真诚修复" → 修补"他人都靠不住"的内在图

### 技术 4:命名分裂
核心话术:"上周的你不信我,今天的你信我。这两个都是你。"

### 技术 5:内在图改写
核心强化:"你今天没有崩——你注意到了吗?"

## 八、临床应用案例

### 8.1 焦虑 × 2 号的客体关系工作

L 女士(32 岁,焦虑 2 号,32 次咨询)

前 16 次咨询(双轴工作):
- 看见"付出=被爱"的等式
- 命名"追-逃"循环
- 训练"不付出地接受"

第 17 次咨询(轨道 C 启动):
- L 带着愤怒来:"我婆婆来我家,只夸她女儿(丈夫的姐姐),根本不夸我"
- 咨询师:"你感受到的是什么?"
- L:"愤怒——我做了那么多,她根本看不见"
- 咨询师:"你注意到了吗?你在愤怒——但你的'老习惯'是'不说出来,继续付出'"
- L:"是的……"
- 咨询师:"说出愤怒,对你来说,意味着什么?"

L 沉默很久。然后说:"意味着我不再有用。如果我'不付出'……我就什么都不是了。"

这是客体关系的核心:
- 来访者自己说出了"我=有用,不用=不存在"
- 这是自体表征
- 咨询师不说"你不应该这样想"——这是认知工作
- 咨询师说:"这听起来像你很早就学到的。你能想到这是什么时候学到的吗?"

L:
> "我妈妈……她生我的时候得了产后抑郁。我从小就是'乖孩子'——我做家务、我照顾弟弟、我让她开心……我必须有用。否则她不会看我一眼。"

这是早年形成的"自体表征"——"我必须有用,否则我看不见"。

客体关系工作:
- 命名这个表征("你从那时候起就学会了'我必须有用'")
- 不评判("那是你活下来的方式")
- 给"新经验"("你今天'没用',但我仍然看见你、关心你")
- 让来访者身体记住这个新经验

第 18-32 次(继续轨道 C):
- L 开始"允许自己不付出"
- L 开始"看见自己的愤怒"
- L 开始"接受'我不需要有用'"

32 次咨询结束时:
> L:"我不再'必须'对他有用——我选择对他有用。这不一样。"
> L:"我允许自己有'不付出'的时候——我没有'消失'。他仍然在。"

这是客体关系修补的成果——不是"我必须有用"被"教育掉",而是"我发现我不需要有用也仍然存在"。

### 8.2 混乱 × 4 号的客体关系工作

C 女士(28 岁,混乱 4 号,40 次咨询)

关键转折(第 18 次,前面已写过):
- C 问咨询师:"你真的关心我吗?"
- 咨询师:过度回应——说"我在这里爱你"
- 咨询师当晚觉察:这太过了——"我爱你"是专业边界外的话
- 第 18 次咨询:咨询师修正边界——"我关心你——这是'专业的关心'。你父亲把'以你为由的自我'给你,这不是爱。我给的不是那种。"

这是破裂-修复的早期:
- 破裂:"我在这里爱你"(咨询师过度)
- 修复:"我关心你——这是专业的关心"(咨询师修正)
- 新经验:"咨询师也会'失误',但他仍然关心我,而且能修复"

C:
> "你的'我爱你'太过了——但你的修正让我安心。你知道吗?我的父亲从来没修正过他的'爱'。他一直说'为你而活'。"
> "你是第一个承认'我可能错了'的人。这让我感觉到——你真的在关系里。"

这是客体关系修补的关键——C 内在图"他人都不会修复"被改写为"有人会修复"。

## 九、客体关系的"陷阱"

### 陷阱 1:把"内省"变成"自我攻击"

> 来访者被告知"你的内在图是 X"——然后开始自我攻击:"我有这样的内在图,我是不是很糟糕?"

应对:
- 先承认这是"早年的智慧"("你当时用它活下来")
- 再命名"现在它不再需要"("现在你是成年人,你有新的选择")
- 避免让来访者羞耻于自己的内在图

### 陷阱 2:咨询师"扮演"来访者的内在图

> 来访者内在图:"他人都靠不住"
> 咨询师为了让来访者体验"被靠得住",过度付出、过度包容

应对:
- 咨询师以身作则"可靠"——不是"过度付出",是"说到做到"
- 准点开始、准点结束、说到做到——这是"可靠"的真正含义
- "过度付出"反而激活来访者的"他人都靠不住"(因为它"不自然")

### 陷阱 3:把"新经验"做成"说教"

> 咨询师:你今天没崩,这是新经验,你的内在图开始变化了。
> 来访者:(一脸迷茫)

应对:
- 让来访者自己命名新经验("你自己注意到没崩了吗?")
- 咨询师强化("对——这是你看见的,不是我告诉你的")
- 避免咨询师"权威"地说"你变了"——这是重复"我=被打分"的早年经验

## 十、最后的提醒

> 客体关系工作不是:
> - 命名来访者"有病"
> - 揭穿来访者"投射"
> - 教育来访者"应该怎么想"
>
> 客体关系工作是:
> - 看见来访者的内在图(咨询师自己)
> - 命名这个图对来访者的影响(让来访者自己看见)
> - 提供新经验("这次不一样")
> - 破裂后修复(示范"关系可以修复")
> - 让来访者慢慢修补自己的内在图
