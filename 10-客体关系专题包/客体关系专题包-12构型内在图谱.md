# 客体关系专题包 · 12 构型内在图谱

> 核心问题:"这个来访者,他/她的'内在世界'长什么样?"
> 核心工具:每个构型的自体表征 / 客体表征 / 关系模板 / 投射性认同 / 温尼科特缺口 / 破裂点
> 使用方法:概念化来访者时,翻此文档

---

## 一、为什么这份图谱是"必需" 的

### 1.1 概念化的"深度"

双轴概念化能告诉你:
- 来访者怎么做(依恋)
- 来访者为什么(九型)

三轴概念化多告诉你:
- 来访者内在世界长什么样(客体关系)
- 来访者期待咨询师做什么
- 来访者会把咨询师变成谁

### 1.2 治疗师的"未说"任务

在咨询中,治疗师不止"听"和"说"——还在做一件未说的任务:
> 识别并修补来访者的内在图。

要做到这一点,治疗师必须:
- 知道来访者的自体表征(他以为自己什么样)
- 知道来访者的客体表征(他以为他人什么样)
- 知道来访者的关系模板(他以为关系怎么运作)
- 知道来访者的投射性认同(他怎么诱导他人验证他的内在图)
- 知道来访者的温尼科特缺口(他没得到的"足够好的"回应)
- 知道来访者的破裂点(什么会验证他的内在图)

## 二、12 构型的内在图谱(详细)

### 构型 1:焦虑 × 2 号助人型

#### 核心信息
- 自体表征:"我是有用的(因为有人需要我)"
- 客体表征:"他人是脆弱的、需要我的"
- 关系模板:"我付出=他感激=他爱我"
- 投射性认同:投射"无助"→对方真的变得依赖、脆弱
- 温尼科特缺口:被无条件爱("不需要我做事,只是爱我")
- 破裂点:付出未被感激(内在图"我必须有用"被验证)
- 自体客体需求:镜映("看见我的付出") + 孪生("我属于这个家")

#### 早年形成
- 典型早年:"我必须照顾妈妈 / 弟弟 / 爸爸,才能被看见"
- 家庭功能:"我的角色是'照顾者',不是'被照顾者'"
- 典型事件:"我做了 X,妈妈终于笑了" / "我考了 100 分,爸爸才看我一眼"

#### 咨询中识别
- 来访者反复问"我能帮你什么吗?"
- 来访者主动给咨询师"送礼物"(带水、给建议、问私人问题)
- 来访者反复说"我做得还不够好"

#### 投射性认同细节
- 来访者:把"被照顾的需要"压抑到无意识
- 投射给咨询师:"咨询师也很脆弱,需要我"
- 咨询师:真的开始感到"被这来访者照顾" / "被需要"
- 验证路径:"我付出了咨询师也感激 = 我在关系中 = 我存在"

#### 破裂-修复
- 破裂:咨询师拒绝来访者的"礼物"/"照顾" → 来访者感到"被拒绝"
- 修复:咨询师真诚说:"你今天没有照顾我——但我仍然看见你、关心你。这不代表你'没用'——你被爱是因为你是你。"
- 新经验:"我可以不付出,我仍然被看见"

#### 客体关系干预
- 不接受"被照顾"的角色(温和拒绝)
- 命名来访者的"照顾"是自动的("你意识到你刚才在'照顾'我吗?")
- 提供"无条件被看见"的体验("今天你什么都没做,但我仍然在这里")
- 强化"我不付出也被爱"的新经验

#### 整合方向
- 轨道 C 目标:"我可以付出,也可以不付出——两个都是我"
- 整合后:"我有选择地付出" vs 之前"我必须付出"

### 构型 2:焦虑 × 6 号疑惑型

#### 核心信息
- 自体表征:"我是无依无靠的、需警觉的"
- 客体表征:"他人会背叛、惩罚、不可信"
- 关系模板:"我警觉=我安全"
- 投射性认同:投射"不可信"→对方真的变得"不可信"
- 温尼科特缺口:可预测性("他做了他说的,说到做到")
- 破裂点:不一致(咨询师说一套做一套)
- 自体客体需求:理想化 + 孪生("我是这个'安全'团体的一员")

#### 早年形成
- 典型早年:"父母反复变化,我不知道'今天'会怎样"
- 家庭功能:"我的角色是'警觉者',扫描威胁"
- 典型事件:"妈妈答应的事没做到" / "爸爸走时说的话是真的吗"

#### 咨询中识别
- 来访者反复说"你保证吗?"
- 来访者反复"测试"咨询师(取消预约、迟到)
- 来访者反复问"我真的可以信任你吗?"

#### 投射性认同细节
- 来访者:把"被背叛"恐惧压在内在
- 投射给咨询师:"咨询师也会背叛我"
- 咨询师:真的开始"无意中"忘记小细节(因为反移情)
- 验证路径:"咨询师真的也靠不住 = 我必须继续警觉"

#### 破裂-修复
- 破裂:咨询师迟到 / 忘记某事 → 来访者立刻感到"被背叛"
- 修复:咨询师不辩护、真诚道歉 + 主动说"我理解这让你失望"
- 新经验:"咨询师真的也会犯错,但他能修复"

#### 客体关系干预
- 不过度保证("我保证"反而强化"我=需要保证")
- 以身作则"可靠":准点开始、准点结束、说到做到
- 主动"示范破裂+修复"——咨询师故意迟到,然后认真修复
- 强化"从实际学习信任"vs"从保证学习信任"

#### 整合方向
- 轨道 C 目标:"我可以信任也可以不信任——两个都是我"
- 整合后:"我信任自己的判断" vs 之前"我必须反复求证"

### 构型 3:焦虑 × 4 号浪漫型

#### 核心信息
- 自体表征:"我是独特的 / 有缺陷的 / 与众不同"
- 客体表征:"他人是完整的 / 普通的 / 不理解我"
- 关系模板:"我不同=我被找到 / 我被找到=我被爱"
- 投射性认同:投射"被抛弃"→对方真的开始回避
- 温尼科特缺口:被"独特地"看见("不是普通的被看见,是我的独特被看见")
- 破裂点:被"普通"对待("你只看见我'普通'的部分")
- 自体客体需求:镜映("看见我的独特") + 孪生("有人和我一样")

#### 早年形成
- 典型早年:"父母缺席/不理解我,我与他们不同"
- 家庭功能:"我的角色是'被忽略的 / 异类'"
- 典型事件:"我说的话没人在意" / "我的情绪被说'想太多'"

#### 咨询中识别
- 来访者反复说"你真的懂我吗?"
- 来访者反复"测试"咨询师(迟到、不赴约、挑咨询师)
- 来访者反复陷入"被抛弃"的戏剧化

#### 投射性认同细节
- 来访者:把"被抛弃"恐惧压在内在
- 投射给咨询师:"咨询师也会离开我"
- 咨询师:真的开始"走神" / "无意中冷淡"
- 验证路径:"咨询师真的也离开我 = 我果然会被抛弃"

#### 破裂-修复
- 破裂:咨询师无意中冷淡 / "标准化"回应 → 来访者戏剧化"你也不真的关心我"
- 修复:咨询师真诚说"我听到你说'我不真的关心'——这让我难过,因为我真的关心"
- 新经验:"我戏剧化的时候,咨询师没有走——他仍然在"

#### 客体关系干预
- 不进入"救世主"角色(混乱 4 号特别爱投射"被拯救")
- 不否定来访者的"戏剧化"("你太夸张了")
- 示范"整合":"你今天戏剧化,但这也是你——我都接受"
- 强化"你不需要戏剧化,我也看见你"——新内在图

#### 整合方向
- 轨道 C 目标:"我可以戏剧化也可以不戏剧化——两个都是我"
- 整合后:"我不需要独特才被看见" vs 之前"我必须独特"

### 构型 4:回避 × 5 号观察型

#### 核心信息
- 自体表征:"我是被入侵的、应被保护的"
- 客体表征:"他人是入侵我空间的"
- 关系模板:"我退=我安全"
- 投射性认同:投射"入侵"→对方真的开始"压过来"
- 温尼科特缺口:尊重他者("咨询师不压过来")
- 破裂点:咨询师表达"关心" (被来访者体验为"入侵")
- 自体客体需求:理想化("咨询师是强的,不会压我")

#### 早年形成
- 典型早年:"父母侵入我的空间,我'不碍事'最安全"
- 家庭功能:"我的角色是'隐形人' / '不添麻烦者'"
- 典型事件:"妈妈反复问'你在想什么'" / "父母未经允许动我的东西"

#### 咨询中识别
- 来访者沉默很久
- 来访者反复说"我自己可以"
- 来访者反复用"理性化"代替"感受"

#### 投射性认同细节
- 来访者:把"被入侵"恐惧压在内在
- 投射给咨询师:"咨询师也会入侵我"
- 咨询师:真的开始"压过来"——多问一句、靠近一点、问感受
- 验证路径:"咨询师真的也压过来 = 我必须更退"

#### 破裂-修复
- 破裂:咨询师过度热情 / 反复问感受 → 来访者感到"被入侵"
- 修复:咨询师主动说"我注意到我刚才问了你太多——你可以说'不'——我接受"
- 新经验:"我说'不',咨询师没生气"

#### 客体关系干预
- "不压过来"姿态:咨询师不主动靠近、不表达过多关心、不追问感受
- 不要求"打开":"我不需要你打开什么"
- 接受沉默:5 号可能咨询中沉默 10 分钟——咨询师不忙打破
- 提供"退出选项":随时可以说"我不想谈这个"
- 强化"咨询师不会压过来"——新内在图

#### 整合方向
- 轨道 C 目标:"我可以靠近也可以独立——两个都是我"
- 整合后:"我选择靠近" vs 之前"我必须远离"

---

### 构型 5:回避 × 8 号挑战型

#### 核心信息
- 自体表征:"我是脆弱的、需保护的"
- 客体表征:"他人会利用我的脆弱"
- 关系模板:"我强=我不会被伤害"
- 投射性认同:投射"利用"→对方真的开始"怕他"
- 温尼科特缺口:可受控的暴露("在绝对安全下显示软")
- 破裂点:咨询师表现出"弱" (被来访者体验为"可被打败")
- 自体客体需求:理想化("咨询师是强的,不会被我打败") + 孪生("有和我一样强的人")

#### 早年形成
- 典型早年:"父母反复让我看到'软'的代价"
- 家庭功能:"我的角色是'保护者'(妈妈、弟弟、家人)"
- 典型事件:"妈妈哭时没人保护" / "我说'我也怕',被爸爸打"

#### 咨询中识别
- 来访者反复表现"强"
- 来访者反复攻击咨询师
- 来访者反复用"否认脆弱"保护自己

#### 投射性认同细节
- 来访者:把"脆弱"压抑到无意识(8号典型的"反向形成")
- 投射给咨询师:"咨询师也会利用我"
- 咨询师:真的开始"怕"8号 / 真的开始"用" 8号
- 验证路径:"咨询师真的也怕我 / 也想用我 = 我必须更强"

#### 破裂-修复
- 破裂:咨询师无意中"被 8 号打败" / "向 8 号投降" → 内在图"软=被打败"被验证
- 修复:咨询师真诚说"我刚才 '投降' 了——不是因为你强,是因为我 累了——但我仍然在这里"
- 新经验:"我不需要永远强——咨询师也不会因为我'不强'就走"

#### 客体关系干预
- 温柔的坚定——不手软(被视为"他也不够强")、不手硬(被视为"威胁")
- 不"打掉" 8 号的"强"——这是他用了很久的防御
- 提供"能不反击"的体验:咨询师不反击 8 号的攻击
- 示范"是人也受伤":咨询师事后真诚说"那句话伤害到我了"
- 强化"今天你没强,但仍然来了"——新内在图

#### 整合方向
- 轨道 C 目标:"我可以强也可以软——两个都是我"
- 整合后:"我选择强" vs 之前"我必须强"

### 构型 6:回避 × 3 号成就型

#### 核心信息
- 自体表征:"我是有价值的(角色) / 我是失败的"
- 客体表征:"他人是观众 / 裁判"
- 关系模板:"我成功=我值得存在"
- 投射性认同:投射"评判"→对方真的开始"评判"
- 温尼科特缺口:被爱不是因为成就("我不做事,你也爱我")
- 破裂点:咨询师"不认同"成就 (被来访者体验为"我不值得")
- 自体客体需求:镜映("看见我的成就") + 理想化("咨询师是强的")

#### 早年形成
- 典型早年:"父母只看见我的'成就' / '形象'"
- 家庭功能:"我的角色是'成功者' / '门面'"
- 典型事件:"我没考好,爸爸不看我" / "我没得奖,妈妈不开心"

#### 咨询中识别
- 来访者反复谈成就、目标、形象
- 来访者反复说"我没什么" / "我还不够好"
- 来访者反复问咨询师"我做对了吗"

#### 投射性认同细节
- 来访者:把"失败"恐惧压在内在
- 投射给咨询师:"咨询师也会评判我"
- 咨询师:真的开始"看"3号的"门面"而不看"3号"
- 验证路径:"咨询师也看我的门面 = 我必须保持门面"

#### 破裂-修复
- 破裂:咨询师无意中问"你这个项目不是没做好吗" → 来访者感到"被评判"
- 修复:咨询师真诚说"我不是在评判你——我是真的好奇——你怎么看这个项目"
- 新经验:"咨询师没评判我的失败——他想看见我的真实"

#### 客体关系干预
- 不进入"评判"角色(咨询师不"打分")
- 不只"看见"3号的成就——也看见3号的"角色后面"
- 示范"不评判":"你今天没做任何事——我仍然看见你"
- 强化"我不成就,你也爱我"——新内在图

#### 整合方向
- 轨道 C 目标:"我可以成就也可以不成就——两个都是我"
- 整合后:"我选择成就" vs 之前"我必须成就"

### 构型 7:混乱 × 4 号浪漫型

#### 核心信息
- 自体表征:"我是有缺陷的、不完整的、坏的"
- 客体表征:"他人是完整的、好的、能拯救我的"
- 关系模板:"我靠近=我受伤 / 我推开=我安全"
- 投射性认同:投射"被抛弃"→对方真的开始回避
- 温尼科特缺口:被完整接纳("我有缺陷,你也爱我")
- 破裂点:咨询师不一致(上周热情,这周冷静)
- 自体客体需求:镜映("看见我的深渊") + 理想化 + 孪生("我属于这里")

#### 早年形成
- 典型早年:"父母剧烈波动(爱/恨/抛弃/回来)"
- 家庭功能:"我不知道我的 '位置' / 我怕受伤"
- 典型事件:"妈妈反复说爱你 / 又说不要你" / "爸爸反复出现 / 消失"

#### 咨询中识别
- 来访者反复"推-拉"
- 来访者反复戏剧化"被抛弃"
- 来访者反复在"上帝"和"魔鬼"之间切换咨询师

#### 投射性认同细节
- 来访者:把"被抛弃"恐惧压在内核
- 投射给咨询师:"咨询师也会抛弃我"
- 咨询师:真的开始"走" / "冷淡"
- 验证路径:"咨询师真的也走 = 我果然会被抛弃"

#### 破裂-修复(混乱 4 号最关键)
- 破裂:咨询师无意中冷淡 / 迟到 → 来访者立刻戏剧化"你也走了"
- 修复:咨询师真诚说"我没有走——我是在这里——但我接受你的怕"
- 新经验:"我戏剧化的时候,咨询师没走——他能接住我的'怕'"

#### 客体关系干预
- 不进入"救世主"角色(混乱 4 号最爱投射"被拯救")
- 不否定来访者的"戏剧化"
- 示范"整合":"上周你说'你不真的关心',今天你说'你最懂我'——两个都是你"
- 强化"你戏剧化的时候,我仍在"——新内在图

#### 整合方向
- 轨道 C 目标:"我可以戏剧化也可以不戏剧化——两个都是我"
- 整合后:"我信任 '真的'和'假的'都是我"

### 构型 8:混乱 × 1 号完美型

#### 核心信息
- 自体表征:"我是应该被批评的、有缺陷的"
- 客体表征:"他人是会审判我的、严苛的"
- 关系模板:"我完美=我安全 / 我犯错=我被抛弃"
- 投射性认同:投射"审判"→对方真的开始"批评"
- 温尼科特缺口:被爱不是因为完美("我犯错,你也爱我")
- 破裂点:咨询师"看见"来访者的"错"(被体验为"审判")
- 自体客体需求:镜映("看见我的'对'") + 理想化("咨询师是完美的")

#### 早年形成
- 典型早年:"父母严苛 / 高标准 / "你应该 做得更好"
- 家庭功能:"我的角色是'应该完美者'"
- 典型事件:"我考 99,爸爸问'1 分去哪了'" / "妈妈说'你能做更好'"

#### 咨询中识别
- 来访者反复自我批评
- 来访者反复"列清单" / "反刍"
- 来访者反复说"我不够好"

#### 投射性认同细节
- 来访者:把"被批评"内化为"内在批评者"
- 投射给咨询师:"咨询师也会批评我"
- 咨询师:真的开始"挑"1号的"错" / "建议"1号"应该"
- 验证路径:"咨询师真的也挑我的错 = 我果然是'应该被批评'的"

#### 破裂-修复
- 破裂:咨询师无意中说"你应该 做得更好" → 来访者感到"被审判"
- 修复:咨询师真诚说"我刚才 说'你应该'——这 是我 的'旧 模式'——我 收回"
- 新经验:"咨询师也有'应该'系统——但他能看见它、修复它"

#### 客体关系干预
- 不进入"评判"角色(咨询师不"挑错")
- 示范"不评判":"你今天犯了一个错——我没评判你——我看见你"
- 强化"我犯错,你仍然爱我"——新内在图
- 命名"内在批评者":"这个声音不是你——是早年学的'我'"

#### 整合方向
- 轨道 C 目标:"我可以完美也可以不完美——两个都是我"
- 整合后:"我选择做得好" vs 之前"我必须做得完美"

### 构型 9:混乱 × 9 号和平型

#### 核心信息
- 自体表征:"我是不存在的、应被忽略的"
- 客体表征:"他人是重要的、应被关注的"
- 关系模板:"我消失=我安全"
- 投射性认同:投射"需要"→对方真的开始"提要求"
- 温尼科特缺口:被主动关注("我没要求,你也看见我")
- 破裂点:咨询师"没看见"来访者 (被体验为"我=不存在")
- 自体客体需求:孪生("我也是这个'重要'团体的一员") + 镜映("我也重要")

#### 早年形成
- 典型早年:"父母忽略我 / 我的需要不重要"
- 家庭功能:"我的角色是'和事佬' / '背景' / '隐形人'"
- 典型事件:"我说'我想'，被忽视" / "家里有冲突,我 帮和"

#### 咨询中识别
- 来访者反复"不主动"
- 来访者反复说"我没什么"
- 来访者反复进入"麻醉" / "解离"

#### 投射性认同细节
- 来访者:把"被忽略"压在内在
- 投射给咨询师:"咨询师也会忽略我"
- 咨询师:真的开始"略过"9号 / "用得更多时间在别的来访者"
- 验证路径:"咨询师真的也忽略我 = 我果然是'不存在'的"

#### 破裂-修复
- 破裂:咨询师无意中说"你的 问题 不 太 重要" → 来访者感到"被验证'不存在'"
- 修复:咨询师真诚说"我刚才 略过 你的 话——对 不起——请 再说 一遍"
- 新经验:"我说 的话,咨询师想 听——不是'不存在'的"

#### 客体关系干预
- 主动"看见"9号——"我注意到你刚才'在' / '说'了一句重要的话"
- 不进入"9号不重要"的角色(咨询师不让9号"自己靠后")
- 强化"你重要"——"你今天来到这里,这 是你的'选择'"
- 示范"不忽略":"你今天说的事,我记得——你不是'不存在'的"

#### 整合方向
- 轨道 C 目标:"我可以在场也可以不在场——两个都是我"
- 整合后:"我选择在场" vs 之前"我必须不在场"

### 构型 10:安全 × 7 号享乐型

#### 核心信息
- 自体表征:"我是有趣的、快乐的"
- 客体表征:"他人是会限制我的、痛苦的"
- 关系模板:"我们有趣=我存在"
- 投射性认同:投射"限制"→对方真的开始"限制"
- 温尼科特缺口:被"深度"地看见("我不有趣,你也爱我")
- 破裂点:咨询师"无聊" (被体验为"我也被无聊")
- 自体客体需求:孪生("有和我一样有趣的人") + 镜映("看见我的'快乐'")

#### 早年形成
- 典型早年:"家庭痛苦,我用'快乐'保护自己"
- 家庭功能:"我的角色是'开心果' / '气氛组'"
- 典型事件:"家里有冲突,我让大家笑"

#### 咨询中识别
- 来访者反复"飞" / 转移话题
- 来访者反复"计划新项目"
- 来访者反复"合理化"痛苦

#### 投射性认同细节
- 来访者:把"痛苦"压抑到无意识
- 投射给咨询师:"咨询师也会让我痛苦"
- 咨询师:真的开始"逼"7号"面对痛苦"
- 验证路径:"咨询师真的也逼我 = 痛苦真的是'限制'"

#### 破裂-修复
- 破裂:咨询师无意中"逼"7号面对痛苦 → 7号立刻转移
- 修复:咨询师真诚说"我刚才逼你面对痛苦——这 是我 的'老 习惯'——我 收回——你 可以选择 '不 面对' 也 是OK 的"
- 新经验:"我选择面对痛苦,咨询师没逼我——这是我的'选择'"

#### 客体关系干预
- 不进入"逼"角色(咨询师不用"面对"为名"逼"7号)
- 允许7号"飞"——"我注意到你在转移——这是OK的——你 可以等等再回来"
- 强化"我也能深入"——"你今天说的'深'的话,我听见了"
- 示范"不限制":"我不会逼你 ——你可以保持 '快乐'——也可以'痛'——两个都OK"

#### 整合方向
- 轨道 C 目标:"我可以快乐也可以痛苦——两个都是我"
- 整合后:"我选择快乐 vs "我必须快乐"

### 构型 12:安全 × 3 号成就型

#### 核心信息
- 自体表征:"我是有能力的、有用的"
- 客体表征:"他人是需要我的"
- 关系模板:"我有用=我被爱"
- 投射性认同:投射"需要"→对方真的开始"依赖"
- 温尼科特缺口:被爱不是因为有用("我不做事,你也爱我")
- 破裂点:咨询师"不"用"来访者(来访者感到"没价值")
- 自体客体需求:镜映("看见我的'能力'") + 孪生("我是'能干'团体的一员")

#### 早年形成
- 典型早年:"父母重视 '能力' / '有用'"
- 家庭功能:"我的角色是'做事者' / '门面'"
- 典型事件:"我没做事,没人看我"

#### 咨询中识别
- 来访者反复"做事" / 给建议
- 来访者反复"用"咨询师
- 来访者反复问"我做对吗"

#### 投射性认同细节
- 来访者:把"无价值"压在内在
- 投射给咨询师:"咨询师也会'用'我"
- 咨询师:真的开始"用" 3号的能力 / "忽视" 3号作为人"
- 验证路径:"咨询师真的也用我 = 我果然是'做事者'"

#### 破裂-修复
- 破裂:咨询师无意中"用" 3号的能力 → 3号感到"被验证'没价值'"
- 修复:咨询师真诚说"我刚才用你的能力——我 收回——你 是人,不是'工具'"
- 新经验:"我不做事,咨询师也爱我——我 是人,不是'能力'"

#### 客体关系干预
- 不进入"用"角色(咨询师不让3号"做事")
- 示范"看见人":"你今天没做事——我 仍然 看见 你"
- 强化"你不是你的能力"——"你是你,不是'能力'"
- 命名"做事"模式:"你今天在'做事'——你注意到吗?你可以'不做事'——这也是你"

#### 整合方向
- 轨道 C 目标:"我可以有用也可以不有用——两个都是我"
- 整合后:"我选择有用" vs "我必须有用"

## 四、12 构型的"共同模式"速查

### 共同模式 1:被"验证"的内在图

所有构型都有"被验证的内在图"——来访者反复验证"他相信的关于自己和世界的故事"。

干预:提供"新经验"——让来访者体验到"不一样"。

### 共同模式 2:把咨询师"变成"他人

所有构型都会在某个时候把咨询师"变成"他早年的人:
- 2号把咨询师变成"需要照顾的人"
- 5号把咨询师变成"入侵者"
- 8号把咨询师变成"被打败的对手"
- 4号把咨询师变成"早年的父母"

干预:识别这个"变成",不"扮演"他,也不"反击"——温和地"是咨询师自己"。

### 共同模式 3:咨询师"被卷进"投射性认同

所有构型都会诱导咨询师"成为"他投射的角色——咨询师自己也会"被卷进"。

干预:5 步识别法(见 `客体关系专题包-投射性识别指南.md`)。

### 共同模式 4:"破裂"提供"修复"机会

所有构型都会在某个时候"破裂"咨访关系(咨询师迟到、忘记、说错话)。

干预:主动修复(见 `客体关系专题包-温尼科特时刻与破裂修复.md`)——这是"新经验"。

### 共同模式 5:"新经验" 需要被"强化"

所有构型的"新经验" 都不会自动留下——必须被强化才能修补内在图。

干预:强化("你今天说了 '深'的话——我听见了")。
