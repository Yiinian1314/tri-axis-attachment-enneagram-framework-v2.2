# 咨询师自我校准 · 耗竭识别与恢复指南

> 谁需要这份文档:所有使用本套件的情感咨询师
> 核心问题:你累 了吗? 你累 的程度是 多少? 你该 怎么办?
> 使用方式:每月做一次"耗竭自检"; 出现"早期信号"时立即做"四步恢复法"

---

## 一、为什么"职业耗竭" 是咨询师最大的隐性风险

### 1.1 咨询师工作的本质

咨询师的工作, 表面是"陪人说话", 实质是:

- 持续 地接住 别人的情绪 (愤怒/悲伤/恐惧/羞耻/绝望)
- 持续 地工作 在别人的"早年" (创伤/丧失/遗弃)
- 持续 地维持 "在场" (尽管你自己也 是人)
- 持续 地不被 来访者"卷进" (哪怕他们反复"测试"你)
- 持续 地做 "治疗" (哪怕你没有"立竿见影" 的反馈)

这种工作, 必然有 消耗。 "咨询师永不 累" 是不 可能的。 关键是"识别" + "恢复"。

### 1.2 职业耗竭的"沉默" 本质

咨询师的"职业耗竭", 最 危险的是它常常 是"沉默" 的:

- 你自己 不知道你在"累"
- 你身边 的人不容易 看见 (你是 "咨询师", 你应该 "会照顾自己")
- 你的专业身份 隐藏 了你的"脆弱"
- 你的"应该" 让你否认 你的"累"

结果: 咨询师经常 是"严重耗竭才 承认"——这时候 已经很难 恢复。

### 1.3 职业耗竭的"代价"

对咨询师自己:
- 慢性疲劳 / 慢性疾病
- 抑郁 / 焦虑 / 解离
- 物质使用
- 自我伤害 / 自杀风险
- 关系破裂
- 意义感危机 / "咨询到底 有没有用"

对来访者:
- 不 "在场" — 咨询变成"形式"
- 不 "看见" — 失去"治疗" 能力
- 被 卷进 — 咨询师自己 的议题污染咨询
- 伦理 风险 — "耗竭" 的咨询师更 可能出现伦理问题

## 二、耗竭的"三阶段" 识别

### 2.1 第一阶段:早期信号(可逆)

这是你必须能 识别的阶段——一旦识别, 可以 通过"自我照顾" 恢复。

#### 身体信号

- [ ] 咨询后特别 累, 需要几个小时 才能恢复
- [ ] 慢性轻度 疲劳, "怎么也休息不过来"
- [ ] 睡眠问题(入睡难/早醒/多梦/反复梦见来访者)
- [ ] 头痛/胃部不适/肌肉紧张/反复感冒
- [ ] 食欲变化(暴食/厌食/反复"忘记" 吃饭)
- [ ] 性欲变化
- [ ] 身体"麻木" / "不敏感"

#### 情绪信号

- [ ] 咨询中开始 "飘" / "麻木" / "烦躁"
- [ ] 咨询中开始 出现反移情 (对某位来访者特别有/没感觉)
- [ ] 咨询后情绪 持续几小时 不能平复
- [ ] 出现"我 怎么也 不好" 的感觉
- [ ] 出现"咨询到底 有没有用?" 的怀疑
- [ ] 出现"我真 的适合 做这行吗?" 的怀疑
- [ ] 出现"我又累 了" 的重复感受

#### 行为信号

- [ ] 开始"批量" 处理来访者, 没有 "在场"
- [ ] 开始"拖延" 咨询记录 / 逃避督导
- [ ] 开始"接受" 越来越多的个案
- [ ] 开始"超载" — 每周个案量超过 30 个
- [ ] 开始"咨询外" 反复"想" 某位来访者
- [ ] 开始"咨询外" 反复"谈" 咨询 (对朋友/家人)
- [ ] 开始"用" 物质(酒精/咖啡因/安眠药)
- [ ] 开始"逃" — 频繁请假/迟到/临时取消

#### 关系信号

- [ ] 私人关系被 咨询"侵蚀"(反复梦见来访者/反复想咨询)
- [ ] 与伴侣/家人/朋友的关系恶化
- [ ] "咨询外" 没有 朋友(因为"我没 时间")
- [ ] "咨询外" 没有 兴趣(因为"我太 累")

#### 意义信号

- [ ] 出现"咨询到底 有用吗?" 的怀疑
- [ ] 出现"我到底 是为 来访者, 还是 为自己 (收入/意义感)" 的怀疑
- [ ] 出现"我已经 不像 之前那么 有热情" 的感觉
- [ ] 出现"我好像 在做 一件'没意义' 的事" 的感觉

#### 早期信号自检表(每月 1 次)

| 类别 | 信号数 (0-5) | 你在 哪条线上 |
|------|---------------|----------------------|
| 身体 | ____ | 0-1 = 健康, 2-3 = 早期, 4-5 = 警戒 |
| 情绪 | ____ | 0-1 = 健康, 2-3 = 早期, 4-5 = 警戒 |
| 行为 | ____ | 0-1 = 健康, 2-3 = 早期, 4-5 = 警戒 |
| 关系 | ____ | 0-1 = 健康, 2-3 = 早期, 4-5 = 警戒 |
| 意义 | ____ | 0-1 = 健康, 2-3 = 早期, 4-5 = 警戒 |

解读:
- 总 0-5 = 健康, 继续
- 总 6-10 = 早期信号, 启动"四步法"
- 总 11-15 = 早期→中期, 立即找督导/个人咨询
- 总 16-25 = 中期信号, 必须 调整执业, 严重者暂停

---

### 2.2 第二阶段:中期信号(需要专业介入)

早期信号没被识别 + 没处理 = 中期信号。

#### 身体信号(中期)

- 慢性严重 疲劳, "我怎么 也休息不过来"
- 反复出现的身体疾病(胃溃疡/偏头痛/慢性背痛/慢性失眠)
- 身体"麻木" 升级 — "我感 受不 到自己"
- 出现"解离" 现象
- 出现" 现实感" 丧失
- 出现"频繁" 的医疗就诊

#### 情绪信号(中期)

- 持续 的"麻木" / "没感觉" / "空了"
- 持续 的"烦躁" / "易怒"
- 持续 的"焦虑" / "害怕"
- 持续 的"悲伤" / "无望"
- 出现"我 真的 想死" 的念头
- 出现"我 真的 想伤害 自己" 的念头

#### 行为信号(中期)

- 出现重大 工作失误(咨询时间错误/记录错误/伦理问题)
- 出现反复 取消咨询/迟到/早退
- 出现反复 与来访者咨询外 联系
- 出现物质 使用升级(酒精/药物)
- 出现"逃" 升级 — 长期请假/想辞职
- 出现人际冲突 升级(与同事/督导/来访者)

#### 关系信号(中期)

- 伴侣关系严重 破裂(分居/离婚)
- 家庭关系严重 破裂
- "咨询外" 完全 孤立
- 出现"对人 不信任" 的感觉

#### 意义信号(中期)

- "咨询到底 是什么?" 出现根本性 怀疑
- "我这 一生做 了什么?" 出现存在性 危机
- 出现"我要换 行业" 的认真 想法

#### 中期信号自检表

如果 你有 中期信号任何 1 条:
- 必须 立即 找个人咨询 / 督导
- 必须 调整 个案量(减少/转介/降低频率)
- 必须 认真考虑暂停 执业

---

### 2.3 第三阶段:晚期信号(严重, 必须立即停止执业)

中期信号没被处理 = 晚期信号。

#### 晚期信号

- 出现严重 的抑郁 (持续 2 周以上)
- 出现严重 的解离 / 现实感 丧失
- 出现自杀 念头/计划
- 出现自我伤害 行为
- 出现物质 滥用(酒精/药物)
- 出现来访者 投诉/伦理问题
- 出现重大 的人际/职业危机
- 出现精神 症状(幻觉/妄想/严重焦虑发作)

#### 晚期信号自检表

如果 你有 晚期信号任何 1 条:
- 必须 立即 暂停执业
- 必须 立即 寻求精神科 评估
- 必须 立即 找资深 督导/治疗师
- 必须 通知专业 机构/协会(如果需要)

## 三、耗竭恢复的"四步法"

### 3.1 概述:Stop, Pause, Reset, Rebuild

```
识别(Stop) → 暂停(Pause) → 重置(Reset) → 重建(Rebuild)
   ↓             ↓             ↓              ↓
 看见"我累"   暂时停下     找到"我"     重新开始
```

### 3.2 第一步:识别(Stop) — "我承认 我累了"

最 难的一步。 咨询师最难 承认"我累了"。

#### 关键原则

- 承认 "我累了" 不是 "我不够 好"
- 承认 "我需要 恢复" 不是 "我应该 自己搞定"
- 承认 "我有 耗竭信号" 不是 "我有 病"

#### 怎么做

1. 看 你的"早期信号自检表"
2. 承认 你有 信号——不否认
3. 接受 "我需要 关注我自己"——不羞愧
4. 承诺 "我会 做点什么"——不拖延

#### 反面教材

- ❌ "我只 是工作太 多, 减少 一点就好 了"
- ❌ "我是 咨询师, 我应该 会自己 处理"
- ❌ "我已经 这么累 了, 我没 时间处理' 我累 了' 这件事"
- ❌ "我没 病, 我不 需要任何 东西"

### 3.3 第二步:暂停(Pause) — "我暂时 停下"

短 期、中期、长期 的暂停都需要。

#### 短期暂停(每天/每次咨询后)

- 每次咨询后, 至少 留30 分钟 给自己
  - 不 立刻做下一 个来访者
  - 不 立刻写记录
  - 不 立刻做行政工作
  - 做让你 恢复的事 — 散步/喝水/发呆/听音乐
- 每天留 至少1 小时 给"咨询外" 的事
  - 运动/朋友/家人/兴趣/休息

#### 中期暂停(每周/每月)

- 每周至少 留1 天 完 全不 做咨询
  - 不 写记录
  - 不 思考来访者
  - 不 看专业书
  - 完 全做 "你不是 咨询师" 的事
- 每月留 至少1 个 "完 全休息 日" (周末)
  - 跟咨询完全 隔绝
  - 跟"咨询师身份" 完全 隔绝

#### 长期暂停(每年)

- 每年至少 留2-4 周 的"长假"
  - 真 正不 做任何咨询工作
  - 不 跟来访者联系
  - 不 跟同行讨论来访者
  - 真 正做 "你自己" 的事 — 旅行/学习/家人/兴趣/休息

#### 关键原则

- 暂停不是 "懒"
- 暂停不是 "逃避"
- 暂停是 "专业责任" — 你必须 保持"在场" 的能力, 暂停是 为了"继续在场"

### 3.4 第三步:重置(Reset) — "我找 到' 我'"

核心问题: 你的"咨询师身份" 是不是已经 占领了你整个 身份?

#### 自检:你的"身份" 清单

写 出你的所有 "身份":

1. 咨询师
2. 朋友
3. 伴侣
4. 父母
5. 子女
6. 兄弟姐妹
7. 同学/校友
8. 兴趣爱好者(运动/艺术/阅读/...)
9. 某种"身份"(公民/居民/某组织成员)
10. 你 是什么?(除了上述)

如果 你只 是"咨询师" — 严重警告。
如果 你只 是 2-3 个身份 — 警告。
如果 你有 5+ 个身份 — 健康。

#### 怎么"重置"

1. 找 你的"个人咨询" — 优先
2. 找 你的"督导" — 必须
3. 找 你的"咨询外" 关系 — 朋友/家人/兴趣
4. 重 新检视 你的"个案量"
5. 重 新检视 你的"咨询师身份" 是不是太大

#### 关键原则

- 你不 是"咨询师" — 你是"做咨询师工作的 人"
- 你不 等于"你的来访者" — 你是"陪来访者的 人"
- 你也 是"你自己" — 独立于"咨询师" 的"你"

### 3.5 第四步:重建(Rebuild) — "我重 新开始"

重置 之后, 重建 一个"不 容易耗竭" 的工作模式。

#### 重建的 5 个维度

##### 维度 1:个案结构

- 重 新评估 你的"承载力"
  - 每周安全 个案量:15-25 个 (取决于个案复杂度)
  - 每周警戒 个案量:超过 30 个
  - 每周健康 个案量:15-20 个 + 个人咨询 + 督导
- 调 整你的个案结构
  - 减少复杂 案例(危机/创伤/双相)
  - 调整频率 (从每周 1 次 → 每两周 1 次)
  - 转介 "不 是你的"案例

##### 维度 2:支持系统

- 个人咨询 — 必须 定期
- 督导 — 必须 定期
- 同行 小组 — 建议 定期
- 专业 社群 — 建议 参与
- 家人/朋友 — 必须 维护

##### 维度 3:咨询外的"你"

- 运动 — 每周 3+ 次
- 睡眠 — 每晚 7-8 小时
- 饮食 — 健康规律
- 兴趣 — 至少 1 个
- 关系 — 至少 2-3 个亲密 关系

##### 维度 4:专业成长

- 持续 学习 — 培训/读书/研讨
- 持续 反思 — 写作/案例整理
- 持续 教学 — 督导新人/教学
- 持续 输出 — 写作/演讲/分享

##### 维度 5:意义感

- 重 新定义 "为什么做咨询"
- 重 新定义 "咨询对 你意味着什么"
- 接受 "咨询有 局限" — 不 是所有问题都 能被咨询解决
- 接受 "我有 局限" — 不 是所有来访者都 能被"我" 帮

## 四、耗竭的"具体场景" 与应对

### 4.1 场景 1:你开始"飘" / "麻木"

表现: 咨询中你开始 "飘" / "没感觉" / "走神"。

应对:
1. 承认 "我飘了" — 不否认
2. 慢 下来 — 你可以 说"我刚才 走神了, 我回来"
3. 回到 身体 — 感受脚踩在地上/手在椅子上
4. 命名 你刚才 在"飘" 去哪里 了
5. 如果 反复飘 — 找督导 / 个人咨询

### 4.2 场景 2:你开始"享受" 某位来访者

表现: 你特别 想见某位来访者 / 咨询后反复 想她。

应对:
1. 识别 — 这是反移情 的信号
2. 命名 — "我被 这位来访者激活 了什么"
3. 不 行动 — 不 咨询外联系/ 不 给她"特别" 关注
4. 找 督导 — 必 须在下次 咨询前 找督导
5. 考虑 转介 — 如果反复 出现

### 4.3 场景 3:你开始"逃避" 某位来访者

表现: 你特别 不想 见某位来访者 / 咨询中想 快点结束。

应对:
1. 识别 — 这也 是反移情 的信号
2. 命名 — "我被 这位来访者激活 了什么"
3. 不 行动 — 不 取消 / 不 提前结束
4. 找 督导 — 必 须在下次 咨询前 找督导
5. 考虑 转介 — 如果反复 出现

### 4.4 场景 4:你出现"严重" 的身体/情绪问题

表现: 失眠/反复生病/持续抑郁/出现自伤念头。

应对:
1. 立即 暂停执业
2. 立即 找精神科 评估
3. 通知 你的督导 + 关键来访者 的"紧急联系人"
4. 接 受长期 治疗
5. 康复 后再考虑恢复 执业

### 4.5 场景 5:你出现"伦理" 问题

表现: 咨询外联系来访者/泄露来访者信息/与来访者发生性关系。

应对:
1. 立即 停止执业
2. 立即 通知督导 + 专业 协会
3. 接受 调查 + 处理
4. 接 受长期 治疗
5. 重 新申请 执业资格(如果可以)

## 五、耗竭的"预防" vs "治疗"

### 5.1 预防(永远比治疗重要)

日常:
- [ ] 每周至少 1 次个人咨询
- [ ] 每2-4 周 1 次督导
- [ ] 每周1 天 完全不咨询
- [ ] 每天30 分钟 运动
- [ ] 每天7-8 小时 睡眠
- [ ] 每周至少 1 次"非咨询" 社交
- [ ] 每月至少 1 个"小目标" (非咨询)

每月:
- [ ] "耗竭自检表" 1 次
- [ ] "反移情自检" 1 次
- [ ] "个案量评估" 1 次
- [ ] "生活平衡" 评估 1 次

每年:
- [ ] "个人咨询" 不 停
- [ ] "督导" 不 停
- [ ] "长假" 2-4 周
- [ ] "全面自评" 1 次(依恋/九型/未完成议题)
- [ ] "意义感" 重新 定义 1 次

### 5.2 治疗(早期信号出现时立即开始)

一旦 出现早期信号:
- 立即 启动"四步法"
- 立即 找督导 + 个人咨询
- 立即 调整个案量
- 立即 维护咨询外 关系

一旦 出现中期信号:
- 立即 暂停执业(部分)
- 立即 精神科评估
- 立即 长期治疗

一旦 出现晚期信号:
- 立即 完全暂停执业
- 立即 精神科治疗
- 长期 跟进

## 六、给咨询师的关键问题清单

### 6.1 每天问自己

- [ ] 我身体 怎么样?
- [ ] 我情绪 怎么样?
- [ ] 我今天 做了什么 "恢复" 的事?
- [ ] 我有 "自动导航" 的反应吗?
- [ ] 我有 任何反移情 信号吗?

### 6.2 每周问自己

- [ ] 我这周 咨询后多 少时间才恢复?
- [ ] 我这周 有几次 "飘" / "麻木" / "累"?
- [ ] 我这周 有几次 "被卷进" / "反移情"?
- [ ] 我这周 的"咨询外" 是什么?
- [ ] 我这周 有几次 "享受" / "飘" / "失联"?

### 6.3 每月问自己

- [ ] 我这月 的"耗竭" 程度是 多少?
- [ ] 我这月 的"个案量" 是否 超载?
- [ ] 我这月 的"个人咨询" 是否 持续?
- [ ] 我这月 的"督导" 是否 持续?
- [ ] 我这月 的"咨询外" 关系是 什么状态?

### 6.4 每年问自己

- [ ] 我这年 整体如何?
- [ ] 我这年 有什么"成长"?
- [ ] 我这年 有什么"停滞"?
- [ ] 我这年 有什么"耗竭" 的高发期?
- [ ] 我这年 有什么"重大" 的事件(影响咨询)?
- [ ] 我下一年 的"咨询师方向" 是什么?
- [ ] 我下一年 需要什么 资源/培训/支持?

## 七、咨询师的"职业生命周期"

### 7.1 阶段 1:0-3 年(技能期)

- 风险: 技术焦虑, 知识不足
- 预防: 持续学习 + 个人咨询 + 督导
- 耗竭信号: 经常"挫败", 觉得"我不 够好"

### 7.2 阶段 2:3-10 年(反移情期)

- 风险: 反移情暴露, 平行过程
- 预防: 持续自我观察 + 督导 + 个人咨询
- 耗竭信号: 出现"职业疲倦", 觉得"我没 感觉"

### 7.3 阶段 3:10+ 年(整合期)

- 风险: 意义感危机, 整合困难
- 预防: 持续成长 + 教学/写作 + 多元化身份
- 耗竭信号: 出现"咨询到底 是什么", 觉得"我不 知道我 在做 什么"

### 7.4 阶段 4:资深(意义重建期)

- 风险: 倦怠, 退出
- 预防: 终身学习 + 持续"自我校准" + 持续"重置"
- 耗竭信号: 出现"我想 退出", 觉得"我不 想做 了"

## 八、给咨询师的关键提醒

### 8.1 你是"工具", 但你也 是"人"

- 你是 工具 — 你需要 校准/维护
- 你也 是人 — 你有 需求/情绪/局限
- 你不是 "超人" — 你会 累/会错/会失败

### 8.2 "咨询师" 不是你的 全部身份

- 你有 朋友/家人/兴趣/学习
- 你有 "咨询外" 的"你"
- 你的 身份是 "多" 的, 不 是"一" 的

### 8.3 "累" 不是 "软弱", "累" 是 信号

- "累" 告诉 你"我需要 关注"
- "累" 不 是"我不够 好"
- "累" 是 "我需要 恢复"

### 8.4 "恢复" 是 专业责任

- 你必须 保持"在场" 的能力
- "恢复" 是 你保持 "在场" 的唯一 方式
- "恢复" 是 你对 来访者的 责任

### 8.5 "求助" 是 专业,不是 软弱

- "个人咨询" 是 专业责任
- "督导" 是 专业责任
- "求助" 是 你对 来访者的 责任

## 九、最后的提醒

> 你 不是"咨询师" — 你 是"做咨询师工作的 人"。
> "咨询师" 是你的 工具, 不是 你的 全部。
> "累" 是信号, 告诉你 "你 需要 关注"。

> 愿你 识别 自己的耗竭, 接受 自己的需要, 重建 自己的"咨询师 + 人" 的平衡。
> 愿你 持续成长, 持续恢复, 持续成为 "你自己"。
