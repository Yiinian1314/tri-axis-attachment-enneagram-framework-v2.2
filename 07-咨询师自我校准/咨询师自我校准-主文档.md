# 咨询师自我校准模块 · 主文档

> 谁需要这份文档:所有使用本套件的情感咨询师、伴侣治疗师、成长陪伴者
> 核心问题:"咨询师自己"是工具, 工具需要校准。 你有多了解自己的"反移情地图"?
> 使用频率:每季度做一次"自我校准"; 每次重要咨询后做"微校准"

---

## 一、为什么"咨询师自我校准"是这个套件最关键的一环

### 1.1 一个常被忽略的事实

你, 咨询师, 才是这个治疗里最重要的工具。

你的来访者坐在你面前, 带着他/她的"模式" (依恋 + 九型); 但你也不是白板——你带着你的 模式在听他/她。

> 如果你对自己的"模式" 不自觉, 你会反复" 被卷进" 来访者的模式, 还以为是来访者的问题。

最常见的几个"咨询师自己不知道" 的事:

- "我对 这位来访者特别有感觉" ——可能是因为他/她触发了 你自己的某个早年模式
- "我对 这位来访者特别没感觉" ——可能是因为他/她没有 触发你的任何"开关", 你在" 安全区"
- "我对 这位来访者有 莫名的愤怒 / 悲伤 / 焦虑" ——这不是"他/她真 的激 活了我", 是他/她的"模式"激活 了你自己的"未完成议题"
- "我不 想接 这位来访者" ——这是你的"反移情", 不是"他/她不 合适"

### 1.2 自我校准的三个层次

| 层次 | 频率 | 内容 |
|------|------|------|
| 第一层:微校准(每次咨询后) | 每次咨询后 5 分钟 | 这次咨询, 我的情绪/身体怎么样? 我"被卷进" 了吗? 我有什么"反移情" 没处理? |
| 第二层:中校准(每月) | 每月 1 次 | 这个月, 我整体的"职业状态" 怎么样? 我的"耗竭" 程度? 我的"反移情模式" 有没有反复? |
| 第三层:大校准(每季度/半年) | 每 3-6 个月 1 次 | 我自己的"个人议题" 处理得怎么样? 我需要找个人咨询吗? 我的"咨询师身份" 和"我自己" 平衡吗? |

## 二、咨询师的双轴定位:你是谁?

### 2.1 你自己的"依恋风格" 与"九型主型"

最关键的觉察:你也是 有"依恋风格" 和"九型主型" 的人——你不是中立的观察者。

来访者 带着他/她的"模式" 来见你; 你 也带着你的"模式" 去见他/她; 两个"模式" 在咨询室里相遇。

> 两个"系统" 相遇, 不只是来访者的"系统" 在工作, 你的"系统" 也 在工作。

### 2.2 自评:你的双轴位置

请在咨询开始前(也建议每年一次), 完成以下自评:

#### 第一步:你的依恋风格(0-10 自评)

- 焦虑维度:
  - 工作中, 我担心来访者"不爱我"(担心脱落) 的程度: ____ / 10
  - 我需要来访者"喜欢我 / 感激我 / 表达感谢" 的程度: ____ / 10
  - 咨询中, 我对"沉默 / 不回应" 的焦虑程度: ____ / 10
  - 焦虑均值: ____

- 回避维度:
  - 我对"来访者情感太满" 的不舒服程度: ____ / 10
  - 我在咨询中"保持距离" 的倾向: ____ / 10
  - 我对"被来访者依赖" 的不舒服程度: ____ / 10
  - 回避均值: ____

- 定位:
  - 焦虑低 + 回避低 = 安全型 (咨询师的"理想位置")
  - 焦虑高 + 回避低 = 焦虑型咨询师
  - 焦虑低 + 回避高 = 回避型咨询师
  - 焦虑高 + 回避高 = 混乱型咨询师

#### 第二步:你的九型主型

- 我童年时"必须做到什么, 才能被爱/不被抛弃": _______________
- 我最不能忍受自己身上发生什么: _______________
- 我压力很大时, 自动化地会做什么: _______________
- 你的主型: ___号 (1-9)
- 你的翼型: ___号 或 ___号

#### 第三步:你"咨询师" 的"主型自动导航"

你的九型主型, 会让你在咨询中自动化地有特定反应:

| 你的主型 | 你在咨询中容易"自动化"地 |
|----------|----------------------------|
| 1号 | 追求"做对", 严格, 自我批评, 对来访者的"不自律" 不耐烦 |
| 2号 | "喂养"来访者, "被需要" 才觉得自己有用, 难以拒绝 |
| 3号 | 追求"成果", 跟着来访者的"目标" 跑, 难以"不做事" |
| 4号 | 追求"深度", 容易被"戏剧化" 的来访者吸引, 陷入"灵魂伴侣" 移情 |
| 5号 | 保持"距离", 难碰身体, 喜欢"分析", 难以"在场" |
| 6号 | 反刍"我做得对不对", 投射怀疑, 容易被来访者的"焦虑" 触发 |
| 7号 | 飞走, 难以"深入" 痛苦, 喜欢"解决问题" 而非"陪伴痛苦" |
| 8号 | 控制, 强势, 难承认脆弱, 容易被"软弱" 的来访者激怒 |
| 9号 | 跟着来访者的节奏走, 难"主动推动", 难碰冲突, 容易" 拖" |

关键问题:你的"自动化" 模式, 哪些是你意识到的? 哪些是你还没意识到的?

## 三、咨询师-来访者"配对" 的反移情地图

### 3.1 双向"系统" 相遇

> 你 的"系统" + 来访者 的"系统" = 咨询室里实际发生的事

最危险的咨询师"盲区":
- 你不 知道自己的"系统"
- 你以为 咨询室里只有 来访者的"系统"
- 你被 来访者的"系统" 激活了, 但你以 为是来访者的"问题"

### 3.2 4×9 = 36 个"咨询师-来访者" 反移情组合

理论上, 你的依恋(4种) × 你的九型(9种) × 来访者的依恋(4种) × 来访者的九型(9种) = 无数种组合。

但临床经验告诉我们, 以下几个组合最危险(咨询师"被卷进" 的概率最高):

| 你的"系统" | 来访者构型 | 你最可能 "被卷进" |
|------------|------------|------------------------|
| 焦虑2号咨询师 | 任何来访者 | 你会"喂养" 所有来访者, "被需要" 才是你工作的动力 |
| 焦虑6号咨询师 | 任何来访者 | 你会"投射怀疑" 到来访者身上, 反复求证"咨询有用吗" |
| 回避5号咨询师 | 任何来访者 | 你会"保持距离", 来访者觉得"你在, 但你不在" |
| 回避8号咨询师 | 任何来访者 | 你会"强势", 来访者觉得"被控制" / "被攻击" |
| 混乱4号咨询师 | 任何来访者 | 你会"被卷进" 来访者的戏剧化, 陷入"灵魂伴侣" 移情 |
| 1号咨询师 | 焦虑/混乱来访者 | 你的"严格" 会让来访者更焦虑/更混乱 |
| 7号咨询师 | 焦虑/混乱来访者 | 你的"飞走" 会让来访者更焦虑/更混乱 |
| 9号咨询师 | 任何来访者 | 你的"跟着走" 会让咨询"停滞" |
| 3号咨询师 | 任何来访者 | 你的"做" 会让咨询"做事", 而非"在" |
| 4号咨询师 | 任何来访者 | 你的"深度" 会让咨询"戏剧化" |

最危险的配对:
- 4号咨询师 × 4号来访者 = "灵魂伴侣" 移情高发
- 2号咨询师 × 2号来访者 = "互相喂养", 但没人做"治疗"
- 1号咨询师 × 1号来访者 = "互相审判", 关系变成"法庭"
- 8号咨询师 × 8号来访者 = "权力斗争", 没人愿意"软"

### 3.3 反移情地图:12 个构型 × 咨询师常见反应

每个构型的来访者, 都会"激活" 咨询师特定的反应:

| 来访者构型 | 咨询师最可能 的反移情 |
|------------|----------------------------|
| 焦虑2号来访者 | "我必须回应她"; "她这么辛苦, 我不帮她谁帮?"; "我不喜欢她找别的咨询师" |
| 焦虑6号来访者 | "我必须'通过'她的测试"; "我开始怀疑我做得对不对"; "我开始被她拖进反复求证" |
| 焦虑4号来访者 | "她好特别, 我被她的深度吸引"; "我被她的戏剧化带着走"; "我开始投入情绪, 失去中立" |
| 回避5号来访者 | "我跟着她'分析', 而不碰感受"; "我开始感觉累/无聊"; "我觉得她不需要我" |
| 回避8号来访者 | "我被她攻击, 我想反击"; "我被她激怒, 我开始控制"; "我投降" |
| 回避3号来访者 | "我跟着她'做事', 我开始'给方案'"; "我享受她的'成功', 而不'看见她'"; "我开始'用'她" |
| 混乱4号来访者 | "我被她的'推-拉' 带着走"; "我开始投入, 失去'暂时'的边界"; "我享受她'依赖' 我" |
| 混乱1号来访者 | "我跟着她'审判', 我开始'解释'"; "我被她'应该' 带着跑"; "我享受'做对'" |
| 混乱9号来访者 | "我跟着她' 慢' 着, 我开始'没'"; "我被她'麻醉' 着"; "我开始' 也' 没感觉" |
| 安全7号来访者 | "我跟着她'飞', 我开始'快'"; "我享受她'快乐', 而不'看见'她"; "我'不深入' 她的' 飞'" |
| 安全9号来访者 | "我跟着她' 慢' 着, 我也开始'没'"; "我' 帮' 她'决定', 失去'中立'"; "我享受她'配合'" |
| 安全3号来访者 | "我跟着她'做事', 我开始'给方案'"; "我享受她'成功', 我开始'评价'"; "我开始'用'她" |

反移情的"信号":
- 咨询中, 你开始有强烈的情绪(愤怒/悲伤/焦虑/性吸引/无感)
- 咨询中, 你特别想"做" 什么(给方案/ 解释/ 拯救)
- 咨询后, 你特别累/ 特别亢奋, 持续几天 才恢复
- 你开始对 这位来访者"特别在意" / "特别不在意"
- 你开始"期待" / "害怕" 某位来访者

## 四、咨询师的"个人议题" 与"平行过程"

### 4.1 "平行过程" 的概念

> 来访者的"系统" 激活了你自己的"未完成议题" ——这就是"平行过程"。

经典场景:
- 焦虑2号来访者反复"付出" → 触发你早年"被父母期待" 的议题 → 你开始"过度付出"
- 回避5号来访者"封闭" → 触发你早年"被父母忽视" 的议题 → 你开始"也跟着封闭"
- 混乱4号来访者"戏剧化" → 触发你早年"被父母情绪化" 的议题 → 你开始"被卷进"
- 8号来访者"控制" → 触发你早年"被父母控制" 的议题 → 你开始"反击"

关键问题:
> 你对 这位来访者的反应, 是来访者 的"系统" 激活的, 还是你自己 的"议题" 激活的?

这两者经常 一起工作 ——但你必须能区分, 才能做"治疗" 而不是"被卷进"。

### 4.2 你的"未完成议题" 清单

请每年做一次这个清单:

我最容易被 激活的议题:
- "我怕 被人抛弃" → 触发来访者: 反复脱落/不来的来访者
- "我怕 被人攻击" → 触发来访者: 8号/愤怒的来访者
- "我怕 被人忽视" → 触发来访者: 沉默/不说话的来访者
- "我怕 被人评判" → 触发来访者: 挑剔/审判的来访者
- "我怕 被人靠近" → 触发来访者: 黏附/情绪化的来访者
- "我怕 自己不够好" → 触发来访者: 反复质疑/不信任的来访者
- "我怕 自己没价值" → 触发来访者: 反复"没效果" 的来访者

每一条, 你都"必须":
- 承认 它存在 ——不否认
- 看到 它在咨询中怎么 激活
- 处理 它——通过个人咨询 / 督导 / 自我反思

### 4.3 关键原则:咨询师也要做"个人咨询"

没有任何咨询师 能只靠自己 维持长期工作。 "个人咨询" 不是"有需要才做" 的事, 是"必须" 的事。

- 新咨询师(执业 0-3 年):每周 1 次个人咨询
- 中级咨询师(执业 3-10 年):每两周 1 次个人咨询
- 资深咨询师(执业 10+ 年):每月 1-2 次个人咨询

为什么:
- 你的"反移情" 需要 一个安全空间 慢慢消化
- 你的"未完成议题" 需要 一个专业的他者 帮你看见
- 你的"咨询师身份" 需要 一个"你不是 咨询师" 的空间

## 五、咨询师的"职业耗竭" 识别与恢复

### 5.1 职业耗竭的常见信号

早期信号(可逆):
- 咨询中开始 "飘" / "麻木" / "烦躁"
- 咨询后特别 累, 几天 才恢复
- 开始讨厌 某位来访者, 但不愿 承认
- 开始拖延 咨询记录 / 逃避 督导
- 开始"批量" 处理来访者, 没有"在场"
- 私人生活开始被 咨询侵蚀(反复梦见来访者/反复想来访者/反复谈咨询)

中期信号(需要专业介入):
- 出现身体 症状: 慢性疲劳 / 头痛 / 失眠 / 胃部问题 / 频繁感冒
- 出现情绪 症状: 持续的"累" / 持续的"烦" / 持续的"麻木" / 持续的" 空"
- 出现关系 症状: 与伴侣/家人/朋友的关系恶化
- 出现意义 症状: "咨询到底 有没有用?" / "我到底 适不适合做这行?"
- 开始"逃": 想换 工作 / 想长期休假 / 想"逃" 进自己的"世界"

晚期信号(严重, 必须立即停止执业):
- 出现解离 / 现实感 丧失
- 出现严重 的抑郁 / 焦虑
- 出现物质 使用(酒精/药物) 升级
- 出现自我伤害 / 自杀 念头
- 出现来访者 投诉/伦理问题

### 5.2 耗竭恢复的"四步法"

#### 第一步:识别(Stop)
- 承认:"我累了, 我需要 恢复"
- 不 把"累" 当成"不够好"
- 咨询师也 是人, 也 会累

#### 第二步:暂停(Pause)
- 短期: 一次咨询后多休息 30 分钟
- 中期: 每周1 天 完全不 做咨询
- 长期: 每年1-2 次 长的休假(2-4 周)

#### 第三步:重置(Reset)
- 找 个人咨询 / 督导
- 检视 你的"个案量" 是否超过 你的"承载力"
  - 一般: 每周 15-25 个 个案, 是健康 的
  - 超过 30 个 个案/周: 高 风险
  - 低于 10 个 个案/周: 可能意义感 危机
- 检视 你的"咨询师身份" 是不是唯一 你的"身份"
  - 你还是 朋友 / 伴侣 / 父母 / 孩子 / 学生 / 运动者 / 艺术爱好者?
  - 如果不是, 你必须 重建咨询外 的"你"

#### 第四步:重建(Rebuild)
- 恢复 你的"个人咨询" (如果你停了)
- 恢复 你的"督导" (如果你停了)
- 恢复 你的"咨询外" 关系(朋友/家人/兴趣)
- 重新定义 你的"咨询师身份" ——是" 你是" 的一部分, 不 是" 你是" 的全部

### 5.3 "承载力" 的概念

不是"你能做多少个来访者" 的问题, 是"你能有质量 地做多少个" 的问题。

计算你的"承载力":
- 每周个案量
- 每个个案的工作强度(高频危机?长期疗愈?)
- 你的"反移情模式" 是否反复
- 你的"个人议题" 是否处理中
- 你的"个人资源" 是否充足(关系/兴趣/身体/意义)

如果"承载力" 超载:
- 减少 个案量
- 增加 督导 / 个人咨询
- 调整 个案结构(降低某些个案频率)
- 增加 "咨询外" 的"恢复活动"

## 六、咨询师的"咨询外" 平行关系

### 6.1 为什么"咨询外" 重要

咨询是"专业" 关系, 但你不是"机器人"。 你的"咨询外" 关系, 是你的" 安全基地":
- 与朋友 的关系 → 让你"不" 永远是"咨询师"
- 与伴侣 的关系 → 让你"不" 把来访者的"亲密" 当成"你的亲密"
- 与家人 的关系 → 让你"不" 把来访者的"早年" 当成"你的早年"
- 与兴趣 的关系 → 让你"不" 把咨询当成"生活的全部"
- 与身体 的关系(运动/休息) → 让你"不" 长期处于"激活态"

### 6.2 "咨询外" 关系 vs 咨询关系 的边界

咨询师必须有"咨询外" 的"自己":
- 你的朋友 不是来访者
- 你的伴侣 不是来访者
- 你的家人 不是来访者
- 你的咨询师 (个人咨询的) 不是 你的督导, 不 是你的朋友, 是 你的"专业他者"

警惕:
- 与来访者咨询外 联系(微信/朋友圈/聚会) ——严禁
- 把朋友 当成"来访者" (分析他们/给他们建议) ——警惕
- 把伴侣 当成"咨询练习" (在他们身上做技术) ——警惕
- 把个人咨询 当成"督导" (在个人咨询中讲案例) ——注意 (但有些流派是允许的)

### 6.3 咨询师的"专业孤独"

咨询师是"专业孤独" 的职业:
- 你的朋友 不能 听你的案例 (伦理)
- 你的家人 不懂 你的"反移情"
- 你的社会 不理解 你的"听别人痛苦" 的工作

应对:
- 同行 小组 (peer group) ——每月 1-2 次, 分享匿名化 案例
- 督导 ——定期, 处理" 反移情" / " 平行过程" / " 案例概念化"
- 个人咨询 ——定期, 处理" 个人议题" / " 职业耗竭"
- 专业 培训 ——持续学习, 不孤立

---

## 七、咨询师的双轴自我校准流程(操作版)

### 7.1 微校准:每次咨询后(5 分钟)

问自己:
1. 身体: 这次咨询, 我身体 怎么样? 哪里紧? 哪里松?
2. 情绪: 这次咨询, 我情绪 怎么样? 有什么情绪没处理?
3. 反移情: 我有想 "做" 什么(给方案/ 解释/ 拯救) 吗? 这是来访者 的需要, 还是我 的需要?
4. 在场: 我这次咨询真的 "在" 吗? 还是"飘" 着? "飘" 的时候, 我在想 什么?
5. 触发: 这次咨询, 来访者 有什么触发 了我自己的什么 吗?

记录:
- 在咨询记录 末尾, 加一段"咨询师反思"(只给自己看, 或在督导中讨论)
- 不 把"咨询师反思" 写进正式的 咨询记录 (这是你 的, 不 是来访者的)

### 7.2 中校准:每月(30 分钟)

问自己:
1. 整体状态: 这个月, 我整体 的"咨询师状态" 怎么样?
2. 耗竭信号: 我有早期 的耗竭信号吗? (累/ 烦/ 飘/ 麻木/ 身体症状)
3. 反移情模式: 这个月, 我在某类 来访者身上有反复 的反移情吗?
4. 个人议题: 这个月, 我的个人议题 有触发 吗? 处理得怎么样?
5. 承载力: 我的个案量 是不是超过 了我的"承载力"?

行动:
- 如果有任何 "中期信号", 立即找 督导/个人咨询
- 如果"承载力" 超载, 减少 个案量
- 如果"个人议题" 反复触发, 回到 个人咨询

### 7.3 大校准:每季度/半年(2 小时)

问自己:
1. 我的"系统": 我真的 知道我的"依恋风格" 和"九型主型" 吗? 我真的 知道我的"自动化" 反应吗?
2. 我的"议题": 我有 什么"未完成议题"? 它们怎么 触发我的咨询? 我处理 了吗?
3. 我的"耗竭": 我整体 的"职业耗竭" 程度? 我需要 调整吗?
4. 我的"成长": 我过去 这半年, 在"咨询师" 这个身份上, 有什么成长? 有什么停滞?
5. 我的"方向": 我未来 的"咨询师" 方向是什么? 我需要 什么新 的培训/督导/支持?

行动:
- 找 资深督导做个人 督导 (not 案例督导, 是"咨询师个人议题" 的督导)
- 重新评估 个案量
- 重新定义 "咨询师身份" 之外, "你" 还是谁

## 八、咨询师的"成长" 路径

### 8.1 三个成长阶段

#### 阶段 1:技能阶段(0-3 年)
- 关注点: "我会 这个技术吗?" "我做对 了吗?"
- 风险: 跟着"技术" 跑, 忽略"关系"
- 任务: 学会"基本技术", 学会"自我观察"
- 必备: 每周 1 次个人咨询, 每两周 1 次督导, 持续学习

#### 阶段 2:反移情阶段(3-10 年)
- 关注点: "我为什么 这么有感觉?" "我为什么 反复做错?"
- 风险: 被"反移情" 卷进, 忽略"来访者"
- 任务: 学会"识别反移情", 学会"区分自己的议题和来访者的议题"
- 必备: 每两周 1 次个人咨询, 每月 1 次督导, 同行小组

#### 阶段 3:整合阶段(10+ 年)
- 关注点: "我是 谁?" "咨询对我 意味着什么?"
- 风险: 职业耗竭, 意义感危机, 自我身份单一化
- 任务: 学会"咨询师身份" 和"个人身份" 的整合, 学会"持续成长"
- 必备: 每月 1-2 次个人咨询, 每季度 1 次督导, 持续教学/写作/督导他人

### 8.2 持续的"成长资源"

- 个人咨询 ——处理个人议题
- 督导 ——处理职业议题
- 同行 小组 ——横向支持
- 专业 培训 ——纵向学习
- 阅读 ——独立思考
- 写作/教学 ——反向学习(把自己知道的"说出来" 是最好的学习)
- 专业 社群 ——归属感
- 个人 资源(运动/兴趣/朋友/家人) ——避免职业耗竭

### 8.3 警惕"咨询师职业" 的几个陷阱

| 陷阱 | 描述 | 应对 |
|------|------|------|
| "全能" 幻想 | 以为"我能 修好所有来访者" | 接受"我不能 修好所有来访者"; 接受"转介是专业, 不是失败" |
| "拯救者" 情结 | 以为"我必须 救来访者" | 接受"来访者自己 是改变的主体"; 接受"我只能 陪, 不能' 救'" |
| "职业边界" 模糊 | 以为"我应该 对来访者全 心" | 保持专业边界; 接受"我也 是人, 也 有局限" |
| "咨询师身份" 单一 | 以为"我只 是咨询师" | 保持"咨询师只是 我的一部分"; 保持"咨询外" 的"我" |
| "专业 完美主义" | 以为"我必须 做对 每次咨询" | 接受"咨询有 不确定性"; 接受"我做错 是正常 的, 不 是失败" |
| "耗竭" 否认 | 以为"我不会 累" | 接受"累是 信号"; 接受"休息是 责任, 不 是软弱" |

## 九、咨询师自我校准的"红线"

### 9.1 立即暂停执业的信号

- 你对某位来访者有性 吸引 / 爱慕
- 你对某位来访者有强烈的愤怒 / 厌恶, 持续几次 咨询
- 你开始咨询外 联系来访者(微信/电话/朋友圈)
- 你开始泄漏 来访者信息给朋友/家人
- 你开始在咨询中 暴露自己 的私人议题过多
- 你出现严重 的耗竭 信号(身体/情绪/意义)
- 你出现物质 使用问题
- 你出现自我伤害 / 自杀 念头

### 9.2 必须转介来访者的信号

- 你发现自己 有未处理 的"个人议题", 与这位来访者直接相关(例如: 来访者让你想起你的"早年")
- 你发现自己对这位来访者有强烈 的反移情, 持续几次 咨询, 无法 处理
- 你发现自己 与这位来访者有咨询外 的关系(朋友/家人/同事)
- 你发现自己 正在"被卷进" 来访者的"系统", 无法 维持"中立"

转介 = 专业, 不 = 失败。 接受"有些来访者不是 我能做的" 是专业成熟 的标志。

### 9.3 立即寻求专业帮助的信号(咨询师自己)

- 出现严重 的抑郁 / 焦虑 / 解离
- 出现物质 使用升级
- 出现自我伤害 / 自杀 念头
- 出现来访者 投诉/伦理问题
- 出现家庭 危机(离婚/重大丧失)

## 十、关键问题清单(每个咨询师自问)

### 10.1 关于我自己

- 我真的 知道我的"依恋风格" 吗?
- 我真的 知道我的"九型主型" 吗?
- 我真的 知道我的"自动化" 反应吗?
- 我真的 知道我的"未完成议题" 吗?
- 我正在 做个人咨询 吗?
- 我正在 做督导 吗?
- 我的"咨询外" 的"我" 还在吗?

### 10.2 关于我的来访者

- 我对某位 来访者特别 有感觉吗? 这是我 的"反移情" 吗?
- 我对某位 来访者特别 没感觉吗? 这是我 的"防御" 吗?
- 我有某类 来访者让我"反复" 重复"反移情模式" 吗?
- 我对某位 来访者有"咨询外" 的想法 吗? (这是危险 的信号)

### 10.3 关于我的工作

- 我的个案量 是不是超过 了我的"承载力"?
- 我有早期 的耗竭信号吗?
- 我最近 有没有"批量" 处理来访者, 没有 "在场"?
- 我最近 有没有"逃避" 督导 / 个人咨询?
- 我最近 有没有"期待" / "害怕" 某位来访者?

## 十一、最后的提醒

> "咨询师" 不是"你"。 "咨询师" 是"你" 的一部分。 "你" 比"咨询师" 大得多。

> "我能陪来访者去多远, 取决于我曾独自走过多远。 "

> "咨询师" 这个身份, 既是礼物 (我能陪人), 也是负担 (我必须持续 自我校准)。

> 愿你持续 校准, 持续成长, 持续 成为 你自己。
