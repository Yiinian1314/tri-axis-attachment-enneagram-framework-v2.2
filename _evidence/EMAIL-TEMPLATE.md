## 为什么这能作为证据

### 法律依据

- **中国《电子签名法》第 7 条**: 数据电文不得仅因其是以电子形式而否定其法律效力
- **中国《民事诉讼法》第 66 条**: 电子邮件属于"电子数据"证据
- **中国《著作权法》**: 作品完成即自动取得著作权,**无需登记**

### 司法实践

邮件时间戳在**司法实践**中作为**初步证据**的接受度:
- 最高法院 IP 法庭多次采信邮件时间戳
- 关键要求: 邮件服务商是**第三方中立机构**(自己发给自己也算,因为邮件服务器是中立的)
- 证据力等级: **中等**(低于公证,但高于口头陈述)

### 增强证据力的额外动作

1. **多发几个邮箱**: 主邮箱 + 工作邮箱 + 个人邮箱
2. **抄送"权威"邮箱**: 如 sinosphere-archives@googlegroups.com(Google Group 归档邮件可作时间戳)
3. **保存原始 .eml 文件**: 不要只保存网页版,保存原始邮件源文件(含邮件头)

## 进阶: 同一时刻的"三角证据"

如果你想要更强的证据,可以**同一时刻**做这三件事:

| 时刻 | 动作 | 证据 |
|---|---|---|
| T1 | 邮件自存(本指南) | 邮件服务器时间戳 |
| T1+1min | GitHub commit(见 GITHUB-GUIDE.md) | 公开 commit 时间戳 |
| T1+2min | 蚂蚁链/至信链上传 README | 区块链时间戳 |

**三选一**就足够基础保护。**三选二**就超过 95% 个人创作者的保护强度。**三选三**几乎不可被推翻。
