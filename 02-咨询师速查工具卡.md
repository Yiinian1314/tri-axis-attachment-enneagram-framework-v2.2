> 与《01-理论架构与咨询指南》配套使用

> v2.2 升级:从双轴升级到三轴(依恋×九型×客体关系)

## 一、依恋象限速查表

| 象限 | 一句话识别 | 身体信号 | 咨询室信号 | 容易引发的咨询师反移情 |
|------|------------|----------|------------|------------------------|
| 安全型 | "我可以靠近,也可以独立" | 放松、眼神接触自然 | 不太需要"被救" | 咨询师容易觉得"他没问题,为什么来" |
| 焦虑型 | "我可能不够好,他可能不爱我" | 心跳快、喉咙紧、手心出汗 | 频繁确认咨询师、"你下周还在吗" | 咨询师想安抚→过度保护 |
| 回避型 | "我不需要,自己可以" | 肩膀紧、呼吸浅、目光回避 | 准时但不深入,情感话题切换为理性分析 | 咨询师感到被推开→自己也想撤 |
| 混乱型 | "我想要又怕" | 状态不稳、解离迹象、情绪戏剧化波动 | 一次咨询里多次"翻脸" | 咨询师混乱,跟着情绪走 |

## 四、咨询室常见话术库

### 命名激活态
- "你的系统现在在响警报——但这不代表警报是真的。"
- "我注意到,你说这件事时呼吸变了——我们先慢一点。"
- "这一刻,你是在'感受',还是在'反应'?"

### 命名防御
- "这是你2号在自动反应——还是你真的想做?"
- "如果你的'付出'里有一滴是交易,它会怎么改变这件事?"
- "你的'分析'是在帮你,还是在帮你不感受?"

### 触碰核心恐惧
- "如果真的发生你最怕的事,你会变成什么?"
- "你愿意在这个画面里多待一会儿吗?"
- "我想请你允许这个恐惧在,看看它会不会自己变化。"

### 翻译伴侣
- "他'消失'可能不是'不爱你',是他的'系统'在说'我需要喘气'。"
- "你'追'的时候,他的体验是'被爱',还是'被吞没'?"
- "你俩在用不同的'爱的语言'说话——这不等于他不说。"

### 整合工作
- "你的'类型自动导航'今天被我们看见了。下周,你想主动选择一次,而不是被它带着走。"
- "我想请你想象,健康版的你,会怎么做?"

## 五、咨询师自我预警信号

你可能在反移情——如果你发现自己在咨询中:

- 感到异常疲惫 → 可能你在替来访者承接情绪
- 感到异常想救他/她 → 可能在重复早年照顾者的位置
- 感到异常生气或失望 → 可能在被来访者的依恋模式"按住"
- 感到异常冷漠 → 可能在被回避型来访者"传染"
- 感到混乱 → 可能在与混乱型来访者共振

应对:暂停、督导、回到自己的身体。

## 六、单次咨询快速流程卡(45分钟版)

```
0-5    接续 + 今日目标确认
5-15   状态扫描 + 身体觉察
15-25  命名模式(依恋 + 九型)
25-35  触碰核心 / 整合练习
35-40  作业 + 实验
40-45  反馈 + 预约
```

红线:
- 不在激活态下做认知工作
- 不替来访者做决定
- 不在第一次咨询就给"诊断"
- 触碰核心恐惧前,先确认安全基地够稳

## 七、阶段化关键问题清单

### 阶段1(1-3次):建立关系
- [ ] 来访者的依恋象限?
- [ ] 来访者的九型主型?
- [ ] 健康层级?
- [ ] 早年的关键依恋剧本?
- [ ] 咨询目标协商一致?

### 阶段2(4-8次):看见模式
- [ ] 来访者能在发作时先看见再反应吗?
- [ ] 来访者能识别自己的预警身体信号吗?
- [ ] 来访者能区分"情绪"和"事实叙事"吗?

### 阶段3(9-15次):哀悼与重建
- [ ] 来访者能停留在"核心恐惧"中,而非立刻逃?
- [ ] 来访者能对早年照顾者说出未说之话?
- [ ] 来访者能发展"内在安全基地"?

### 阶段4(16-25次):新经验与整合
- [ ] 来访者在咨询外有"健康版"的新行为吗?
- [ ] 来访者能主动选择而非被模式带着走?
- [ ] 来访者对"整合后的自己"有想象吗?

### 阶段5(长期):维护与成长
- [ ] 来访者脱落频率?能自我调节吗?
- [ ] 来访者能将咨询经验迁移到其他关系?
- [ ] 来访者开始"自我主导",而非"防御驱动"?

## 八、紧急情况与边界

| 情况 | 咨询师反应 | 边界 |
|------|------------|------|
| 来访者出现解离 | 接地技术、感官扫描 | 若频繁,转介精神科 |
| 来访者出现自杀意念 | 安全评估、危机干预 | 暴露严重则需住院 |
| 来访者强烈移情 | 命名移情、保持节制 | 不可在咨询外联系 |
| 咨询师自己被强烈激活 | 暂停、督导、自我咨询 | 必要时要转介来访者 |

## 十、最小可用核心(三句话, v1.0)

如果只能记三件事,请记:

> 1. 依恋告诉你来访者在"怎么反应",九型告诉你"为什么这样反应"。
> 2. 触碰核心恐惧前,先确保安全基地够稳。
> 3. 咨询师的工作不是"修理"来访者,是陪他/她重新认识自己,然后主动选择。

## 十一、客体关系速查(本次新增)

### 1. 三个轴的区分

| 轴 | 关注点 | 类比 |
|---|------|-----|
| 依恋 | 来访者怎么做 | GPS 路径 |
| 九型 | 来访者为什么 | 目的地 |
| 客体关系 | 来访者内在世界长什么样 | 地图本身 |

### 2. 5 个核心概念(一句话版)

| 概念 | 一句话 |
|------|--------|
| 内在客体关系 | "我是什么样的" + "他人是什么样的" 的配对表征 |
| 投射性认同 | 来访者把内在部分"放到"你身上,诱导你"成为"那个部分 |
| 分裂 | 好与坏是完全分开的两个(不能容纳灰色) |
| 温尼科特时刻 | "足够好"的回应:不完美但不冷漠,接住但不抢夺 |
| 破裂-修复 | 关系疗愈不来自不破裂,而来自破裂后的修复 |

### 3. 12 构型的内在图谱(一句话)

| 构型 | 自体表征 | 客体表征 | 投射性认同 |
|------|----------|----------|------------|
| 焦2 | 我是有用的 | 他人是脆弱的 | 投射"无助"→对方依赖 |
| 焦6 | 我是警觉的 | 他人是威胁 | 投射"不可信"→对方不可信 |
| 焦4 | 我是独特的/有缺陷 | 他人是完整的 | 投射"被抛弃"→对方回避 |
| 回5 | 我是被入侵的 | 他人是入侵者 | 投射"入侵"→对方压过来 |
| 回8 | 我是脆弱的 | 他人是利用我的 | 投射"利用"→对方退缩 |
| 回3 | 我是有价值(角色) | 他人是观众 | 投射"评判"→对方评判 |
| 乱4 | 我是坏的 | 他人是能拯救我的 | 投射"被抛"→对方回避 |
| 乱1 | 我是该被审判的 | 他人是审判者 | 投射"审判"→对方批评 |
| 乱9 | 我是不存在的 | 他人是重要的 | 投射"需要"→对方提要求 |
| 安7 | 我是有趣的 | 他人是有趣(直到不有趣) | 投射"无趣"→对方无聊 |
| 安9 | 我是配合的 | 他人是重要的 | 投射"冲突"→对方冲突 |
| 安3 | 我是有能力的 | 他人需要我帮助 | 投射"需要"→对方依赖 |

### 4. 5 个快速识别信号(咨询师反移情)

> 如果你对来访者有强烈反应,问这 5 个问题:

1. 停一下 — "我怎么突然这么[情绪]?"
2. 追溯 — "来访者做了什么触发了这个?"
3. 反推 — "如果这是他投射给我的,他内在什么被激活?"
4. 识别投射 — "我是不是在扮演他内在图中的'他者'角色?"
5. 选择回应 — "不进入投射,也不强硬拒绝;命名投射本身"

### 5. 5 个客体关系技术(一句话版)

| 技术 | 一句话 |
|------|--------|
| 投射性认同识别 | "这股情绪是我的还是你的?" |
| 温尼科特时刻 | "沉默 + 眼神 + '我在这里' = 接住" |
| 破裂-修复 | "咨询师示弱 + 真诚修复" 提供"破裂可修复"的内在图 |
| 命名分裂 | "上周你说我是'不好的',今天你说我'最懂你'——这两个都是你" |
| 内在图改写 | "你今天没有崩,你注意到了吗?" |

### 6. 客体关系红旗(看到立即干预)

- [ ] 来访者反复出现"全好/全坏" 快速切换
- [ ] 来访者“测试”咨询师“能接住多少”
- [ ] 来访者在关系中反复确认“你不会抛弃我吧”
- [ ] 咨询师反复出现"被唤醒"的旧伤
- [ ] 咨访中出现不自然的理想化/贬低切换

## 十二、最终最小可用核心(三句话, v2.2 升级)

v2.2 三轴融合 — 顺序铁律:先依恋(给安全) → 再九型(觉察动力) → 最后客体关系(修补结构)

> 1. 三轴(依恋×九型×客体关系)三视角缺一不可 — 整合公式:依恋激活 × 九型动力 × 客体关系结构。
> 2. 顺序铁律:触碰核心恐惧前,先确保安全基地够稳;修补内在图前,先稳定动力。冲突优先级:安全 > 结构。
> 3. 咨询师的工作不是"修理"来访者,是陪他/她重新认识自己、修补内在图、然后主动选择。
