# 构型深度展开 #06:回避型依恋 × 3号成就型

> 临床常见度:中(高功能来访者中常见, 经常晚来咨询, "事业成功但关系失败" 的典型)
> 来访者画像关键词:永远在"做事"、逃避情感、用成就证明价值、伴侣抱怨"你不在"
> 咨询师最容易犯的错:被来访者的"成功" 唬住, 觉得"她过得很好, 没什么问题"

---

## 一、构型核心图谱

### 1.1 内在逻辑

回避依恋 + 3号成就型的组合,本质是:

> 早年学会了"我必须成就, 才值得被爱"
> 成年后用"做事" 替代"感受", 用"成就" 替代"亲密"
> "成功" 不是她的选择, 是她的身份证——但身份证丢了, 她不知道自己是谁

这个组合的来访者, 在咨询中有一个" 矛盾":
- 表面"非常成功"——事业好、收入高、社交广
- 但情感关系"一片狼藉"——伴侣走了, 朋友散了, 跟父母也疏远
- 她的" 成功" 是真的, 但她的" 成功" 代价 是她" 不在" 关系里
- 她经常很晚才来咨询——因为她" 忙"——但其实, 她的"忙" 正是她的" 不在" 的证据

### 1.2 双轴解读

| 维度 | 表现 |
|------|------|
| 回避依恋(高) | 不需要亲密, 自我依赖, 情感去激活, 害怕被"看见真正的自己" |
| 3号主型 | 注意力朝向"目标" 与"形象", 角色扮演, 否认情感, 用成就定义价值 |
| 融合放大 | 回避让3号的"成就" 变成"回避情感" 的工具; 3号让回避的"独立" 变成" 成功的独立" |

### 1.3 健康层级

- 健康层(1-3层): 自我接纳, 真实, 成就与人格并存
- 平均层(4-6层): 高度竞争, 角色扮演, 不允许" 不成功" / " 不完美", 情感冻结
- 不健康层(7-9层): 严重的工作狂, 失去自我, 出现解离, 失败时崩溃

### 1.4 压力方向

3号 → 9号(压力下的解离):
- 平时冲刺/竞争, 压力下"空/麻木"
- "我做了这么多, 但我什么都不是"
- 这一刻的9号反应, 容易让伴侣/咨询师都困惑——"她不是一直很有能量吗?"

3号 → 6号(压力下进一步):
- 平时"自信", 压力下"怀疑/焦虑"
- "我其实不够好" / "他们发现了怎么办"
- 这一方向的来访者, 容易在"被发现" / "失败" 时出现

### 1.5 客体关系维度(v2.2 新增)

核心内在图谱:

| 客体关系要素 | 回避 × 3 号 |
|------------|------------|
| 自体表征 | "我是有价值的(角色)" / "我是失败的(角色后面)" |
| 客体表征 | "他人是观众 / 裁判" |
| 关系模板 | "我成功 = 我值得存在" |
| 投射性认同 | 投射"评判" → 对方真的开始"评判"他 |
| 典型破裂 | 咨询师"不认同"成就 / 看见"角色后面的人" → 内在图验证 |
| 温尼科特需求 | 被爱不是因为成就("我不做事,你也爱我") + 镜映("看见人,不只看见成就") |

为什么"客体关系" 是 3 号咨询的关键:
- 3 号的"我知道但做不到" 体现在——他知道"成就不等于价值"、知道"不用一直做"——但他做不到
- 原因:他的内在图还在说"成就=价值"、"我=我的角色"
- 认知工作改不掉这个结构——必须在关系中修补
- 3 号的修补特别难——因为他的"角色"是他用来避免接触"后面的人"的防御

咨询师"会"进入的 4 个"他者" 角色(避免):

| 投射角色 | 咨询师反应 | 为什么错了 |
|----------|------------|------------|
| "挑剔的观众" | 咨询师挑刺、"你可以做更好" | 验证"咨询师也在评判"→强化"成就=被看见" |
| "盲目的粉丝" | 咨询师过度赞美、"你太厉害了!" | 验证"咨询师也只看成就"→强化"角色=我" |
| "揭露者" | 咨询师强迫 3 号停下、"别做了" | 验证"我不做事 = 被抛弃"→强化"不做事=无价值" |
| "被利用的助手" | 咨询师接受 3 号的"用"(他把咨询当工具) | 验证"我用他=他是工具"→强化"我是做事者" |

咨询师"应该"提供的 4 个"新经验":

| 新经验 | 怎么做 | 修补的内在图 |
|--------|--------|--------------|
| "看见人,不只看见成就" | "你今天没做事 — 我仍然看见你" | "我只值得被看见(因为成就)" → "我本身值得被看见" |
| "被打分也仍被爱" | 3 号做错了 — 咨询师 不减少关心 | "我做错了 = 被抛弃" → "我做错了 = 仍然被关心" |
| "不做事也被接纳" | 3 号没做事 — 咨询师 不催促 | "我不做事 = 无价值" → "我不做事 = 仍然有价值" |
| "咨询师也'做错'" | 咨询师示弱("我也会累") | "我必须做对 = 他们才爱" → "他们也会做错、仍然有价值" |

3 号常见的 5 个"内在图"自查表:

- [ ] "我只值得被看见(因为成就)"
- [ ] "我的价值 = 我的角色"
- [ ] "我不做事 = 不存在"
- [ ] "我的感受 = 浪费时间"
- [ ] "我不能"失败"(失败 = 死) 

3 号的"温尼科特时刻"是什么样:

- 来访者很罕见地说了一句 "我的感受是..."(不是"我做了...")
- 咨询师不要"大惊小怪"(会让 3 号 "又做错了")
- 咨询师只 接住 + 简单说:"我听到了你的感受了。谢谢你告诉我。"
- 不强调"这是进步"(会让 3 号觉得"我又被打分了")

这是3号的"足够好"回应——不打分、不大惊小怪、只"看见"人。

## 二、早期依恋剧本

### 2.1 常见早年经验

| 类别 | 典型情境 |
|------|----------|
| 父母的" 条件性" 爱 | "你考了 100 分, 我才爱你" / "你比别人强, 我才骄傲" |
| 父母的" 投射" | 父母把自己未实现的目标投射给孩子, 孩子必须" 实现" |
| 父母的" 缺席" | 父母身体在, 但情感上" 不在" / 父母长期出差 / 离异
| 早期的" 角色" | 孩子被当成"小大人" / " 家里的希望" / " 不能失败" |
| 兄弟姐妹的" 比较" | 父母总拿孩子跟" 别人家的孩子" 比, 孩子必须" 赢" |

### 2.2 形成的核心等式

> 如果我不成就, 我就不值得被爱
>
> 如果我停下来了, 我就什么都不是

3号回避型来访者最深的信念, 不是"我不需要爱", 而是"被爱, 是因为我有用 / 成功"。这跟2号微妙但关键地不同:
- 2号: "我必须被需要" → 解决方法是"我必须付出"
- 3号: "我必须成就" → 解决方法是"我必须成功"

### 2.3 形成的"内在演员" 模式

3号来访者的注意力结构, 是永恒的"内在演员":
- 永远在"演" 一个" 成功 / 完美 / 自信" 的角色
- 极少" 在" 真实的自己
- 真实的自己, 是"我不知道 我想要什么" / "我不知道 我是谁" / "我很累"
- 演" 太久了, 她自己也 不知道哪个是真的——这是3号最深的不安

咨询师的关键识别:她在咨询中, 经常呈现" 成功 / 自信" 的形象——但这种"成功 / 自信", 是她的"角色", 不是"她"。

### 2.4 形成的"做 - 感受" 隔离

3号来访者经常把"做事" 当成"感受" 的替代:
- "我做了 X, 我应该 高兴" ——但她真的 高兴吗? 她不知道
- "我必须 继续做 Y" ——为什么? 因为停下来, 就要"面对" 自己
- "我现在没时间 感受" ——她的"没时间" 正是她的" 不敢有时间"

## 三、典型临床表现

### 3.1 在亲密关系中

关系早期:
- 看起来"有魅力" / "有目标" / " 有未来"
- 伴侣觉得" 她/他 好棒, 跟别人不一样"
- 但其实, 她的" 魅力" 是她的" 角色" ——她真的 不知道" 她" 是什么样
- 关系快速进入" 形象配对" 的状态——两个" 成功的人"

关系深入后(形象 vs 真实期):
- 伴侣开始"看见" 她: "你真的 快乐吗?" / "你真的 想做这个吗?"
- 她的反应是" 是啊" / " 怎么? 你质疑我?"
- 她真的 不知道答案——她演 得太久了
- 关系变成" 形象对形象" ——没人真的" 在"

关系破裂时:
- 她可能" 理性" 地结束——" 我们的目标不一致"
- 她的" 理性", 是她的" 角色" 在工作
- 真正的她, 可能正在" 空" / " 麻" / " 解离" 中
- 但她不 让任何人看见这一面

### 3.2 在咨询中

前3次: 她的" 成功" 表现
- 准时, 穿着得体, 表达清晰
- 描述问题" 客观" / " 有目标" / " 想要解决"
- 期待咨询师" 给个方案" / " 帮我达成目标"
- 警惕: 她的" 想要解决" 是她的" 3号" 在工作, 她的" 真的痛苦" 还在" 藏" 着

她对" 情感" 的回避:
- 问她" 你现在什么感觉", 她可能说" 我觉得…… 应该…… 接下来……"
- 她的" 觉得" 经常跟着" 应该" / " 必须" / " 接下来"
- 真正的感受, 经常被" 应该" 盖住了
- 问她" 你害怕什么", 她可能说" 我不怕" / " 我已经处理好了" ——这是她的" 角色"

她对" 失败" 的恐惧:
- 任何暗示" 你失败了" 的话, 都会激活她的" 3号" 防御
- 咨询师不能 直接说" 你失败了" ——这会让她瞬间关闭
- 咨询师可以 温和地问: "你最近, 有没有' 不够好' 的感觉?"

咨询师的关键识别:
- 警惕自己"被她的成功 唬住"——你可能觉得"她过得很好, 没什么问题"
- 警惕自己"被她的目标 带着跑"——你可能开始跟她" 解决问题", 而不是" 看见她"
- 警惕自己"觉得她不需要我"——她的" 不需要" 是她的" 3号" 在工作
- 核心姿态:温和, 稳定, 不被她" 拉着跑", 也不" 否定" 她的成就——成就本身不是问题, 问题是"用成就替代情感"

### 3.3 身体信号

| 激活时 | 长期慢性 |
|--------|----------|
| 心跳快(总在"赶" ) | 慢性疲劳(身体透支) |
| 胃部不适(压力 + 不规律饮食) | 失眠(停不下来) |
| 肩膀/脖子紧 | 慢性疼痛(身体在" 喊" ) |
| 难以呼吸(被" 角色" 压着) | 进食问题(用" 吃" / " 不吃" 调节) |

## 四、咨询路线图

### 4.1 阶段1:建立关系与评估(1-3次)

关键风险:
- 来访者会用"成功 / 目标" 来保持距离——咨询师可能跟着" 解决问题"
- 咨询师必须慢下来, 把节奏从"做事" 转到"感受"
- 重要原则:不否定她的成就, 但邀请她看看"成就下面"

必须做:
- 第一次: 慢一点, 问她" 你来这里, 带着什么感觉?" ——不是" 你有什么目标", 是" 什么感觉"
  - 她可能说" 我想……" / " 我觉得应该……" ——没关系, 慢慢引导到" 感受"
  - 这次咨询的" 基调", 是" 安全, 不催, 慢慢来"
- 第二次: 收集 ECR-R + RHETI, 但不要"只"做评估 ——评估是入口, 不是目的
  - 让她慢慢看见自己的" 模式": " 我发现, 你在' 描述目标' 之前, 身体其实先有反应"
- 第三次: 反馈诊断, 特别要包含" 你的' 成就' 是真的, 但' 成就' 也是你的保护"
  - 让她知道: 你的" 成就" 不是病, 但你也可以" 停下来"

咨询师姿态:
- 慢, 稳, 不催
- 不否定她的成就——"你的成就真的 很厉害, 我能看见"
- 但慢慢 邀请她看"成就下面"——"你的成就, 在保护什么?"
- 关键: 当她说" 我已经处理好了", 温和地: " 我听你说' 处理好了'——但我好奇, ' 处理' 跟' 放下' 是一回事吗?"

关键话术:
> "你说' 我觉得应该……'——我猜, ' 应该' 是你经常用的词——我想问问, 如果没有 ' 应该', 你想要什么?"
>
> "你说你' 不需要停下来' ——我猜, ' 停下来' 对你来说, 等于' 面对' ——面对什么?"
>
> "你的成就真的 很厉害——但我注意到, 你谈成就的时候, 身体是紧绷的——你的身体, 在说些什么?"

### 4.2 阶段2:看见模式(4-8次)

核心技术:
- "案例回放 + 身体的邀请"
  - 跟她做案例回放时, 不断回到身体: "那一刻, 你身体哪里?"
  - 她可能说" 我觉得应该……"——不催, 但继续: " 你的' 应该' 下面, 身体的反应是什么?"
- "应该" vs "想要" 分离
  - "你说' 我觉得应该做 X'——如果没有 ' 应该' 这个词, 你真的 想做 X 吗?"
  - 让她看见: 她的" 应该" 是她的" 角色" 在工作
- "成就" 命名
  - "你说' 我必须继续 努力'——我猜, ' 停下来' 对你来说, 是危险 的——这个危险是什么?"

关键话术:
> "你说' 我做到了 X, 我应该高兴'——但我真的 想知道, 你真的 高兴吗?"
>
> "你说' 我已经处理好了'——我猜, ' 处理' 是你的方式, 但' 感受' 可能还没——你愿意陪我们, 慢一点, 看看' 没处理' 的部分吗?"
>
> "你说' 我必须 继续努力'——如果真的 停下来, 你猜, 你会变成什么?"

咨询师反移情预警:
- 感到"被她的成功 唬住"——你可能觉得" 她没问题, 我帮不上什么"
- 感到"被她的目标 带着跑"——你可能开始" 跟她一起解决问题", 而不是" 帮她看见"
- 感到"我应该给她方案"——你可能进入" 顾问" 模式, 而不是"治疗师" 模式
- 感到"被她的' 不需要' 推开"——你可能觉得" 她不需要我"
- 警惕:这些反移情, 正是她伴侣/家人长期感受到的 ——你的反移情, 是工作的入口

### 4.3 阶段3:哀悼与重建(9-15次)

核心工作:触碰"成就" 的代价——"我不知道我是谁"

关键技术1:空椅子(对早年" 必须成就" 的父母/情境)

让她对那个"让她必须成就" 的情境说话:
- "那个 7 岁的你, 第一次发现' 我没考好, 妈妈/爸爸 就不那么高兴' 的时候, 你有什么感觉?"
- 让她在那个" 不被爱(因为不够好)" 的情绪里多待

关键技术2:身体的工作

这一阶段, 咨询师的工作慢慢从"谈话" 转向"身体":
- 在她描述时, 注意她的肩膀/呼吸/姿势
- "我注意到, 你说' 我很好' 时, 你的肩膀是紧的——能慢一点, 让你的身体, 跟上你的故事吗?"
- 不催她" 必须哭", 但让她知道"哭是OK的"
- 慢, 稳, 不催——她的身体, 需要很长时间才" 敢" 打开

关键技术3:核心恐惧暴露

> "如果——我是说如果——真的你' 什么成就都没有' 了, 不是暂时的, 是永久的, 你会变成什么?"

她的第一反应可能是" 那我就什么都不是"——这是她最深的不安。让这个真实 的恐惧, 在咨询室里完整地存在, 不要急着安抚。

关键技术4:"停下来" 的体验

让她在咨询中小剂量地 体验"我停下来, 但世界没塌":
- 咨询中: "今天, 我们不 谈目标, 不 谈方案——我们就聊聊' 你今天怎么样' ——可以吗?"
- 这种" 不做事" 的体验, 慢慢松动她的" 我必须做事 = 我存在" 等式

咨询师自我校准:
- 这一阶段, 你的" 想帮她解决问题" 会特别强——你可能想" 给她方案"
- 督导中讨论: 我是否在某次咨询中, " 跟着她" 一起" 做事" 了?
- 警惕"咨询师 - 来访者" 之间的" 顾问 - 客户" 移情——3号来访者最容易把咨询师拉入" 解决问题" 的关系

### 4.4 阶段4:新经验与整合(16-25次)

核心工作:让她体验" 做事" 和" 存在" 可以并存

关键技术1:"小剂量" 停下来

- 不是一次性让她" 停下来"
- 而是小剂量地, 慢慢体验" 我不做事, 我也存在"
- 咨询中: "今天, 你愿意不 谈目标, 谈一件你最近想 做但没 做的事吗?"
- 她可能说" 我没什么想做的"——不催, 但继续: " 如果真的 必须给一个, 是?"

关键技术2:咨询师的" 不做事" 自我暴露

- 跟她分享"咨询师也有' 不做事' 的时刻"
- 不是"专业报告", 而是"作为一个人, 我也 有' 不想' 的时候, 我也 允许自己' 不想' "
- 让她看到" 强" 的人也是"有' 不做' 的时候" ——这松动她的" 做事 = 价值" 等式

关键技术3:"健康版 3号" 想象

> "我请你想象——健康版的 3号, 是怎样的? 她仍然有目标, 但她也 能' 不' 有目标。她仍然追求, 但她也 能' 不' 追求。她' 做事' 是因为她想, 不是因为她必须。她是怎么做到这点的?"

整合方向:
- 3号 → 6号: 学会" 允许自己害怕", 学会" 允许自己不确定"
- 回避 → 安全: 学会" 停下来, 不被吞没"

### 4.5 客体关系阶段(本次新增,v2.2 升级)

这一阶段在 16-25 次期间与 4.4 并行,主要修补 3 号的内在图。

#### 4.5.1 核心目标

不是"让来访者不做事"——是"让不做事也有价值"。

3 号的内在图 修补清单:
| 旧内在图 | 新经验 | 咨询师的回应 |
|------------|----------|---------------|
| "我只值得被看见(因为成就)" | "我本身值得被看见" | "你今天没做事 — 我仍然看见你" |
| "我的价值 = 我的角色" | "我不等于我的角色" | 咨询师 不只回应"角色"(看见角色后面的人) |
| "我不做事 = 不存在" | "我不做事 = 仍然有价值" | 3 号没做事 — 咨询师 不减少关心 |
| "我的感受 = 浪费时间" | "我的感受 = 值得被听见" | 3 号说"我的感受是..." — 咨询师 认真听 |

#### 4.5.2 3 号专属客体关系技术

技术 1:看见人(不只看见角色)

> 咨询师: "你今天说了这个项目 — 但我更想听你的感受。你是怎么看这个的?"
> (3 号习惯说"我做了...";咨询师邀请他说"我感觉...")

技术 2:不打分(不把 3 号"打分")

> 3 号说: "我这次考试没考好..."
> 咨询师: "你没考好 — 这是一件事。我不会因此 '打分'。你是你 — 不是你的考试。"
> (3 号的内在图"做错 = 被打低分"被修补为"做错 = 仍然是我")

技术 3:接住"不做事"(不促进、不减少关心)

> 3 号: "这周我什么都没做..."
> 咨询师: "你什么都没做 — 你今天来到这里 — 这是一件事。你是你 — 不是你的 '做事'。"
> (3 号的内在图"不做事 = 无价值"被修补为"不做事 = 仍然是我")

技术 4:破裂-修复(3号的破裂常是"被打低分")

> 咨询师: "我刚才说了'你这个项目可以做得更好' — 这是我的'挑刺' — 对不起。我不应该这样说。你是你 — 不是你的项目。"

#### 4.5.3 3 号最需要的 3 个 "新经验"

| 新经验 | 修补的内在图 | 咨询师的关键动作 |
|----------|------------------|---------------------|
| "看见人" | "我只值得被看见(因为成就)" | 不只回应"角色"(看见角色后面的人) |
| "打分也被爱" | "我做错了 = 被抛弃" | 3 号做错了 — 咨询师 不减少关心 |
| "不做事也有价值" | "我不做事 = 无价值" | 3 号没做事 — 咨询师 不催促 |

## 五、咨询师反移情与督导要点

### 5.1 常见反移情

| 你的反应 | 背后的反移情 | 应对 |
|----------|--------------|------|
| 感到"她没问题" | 你被她的"成功" 唬住了 | 她的" 成功" 正是她的" 防御" |
| 感到"被她的目标 带着跑" | 你的" 想解决问题" 模式 | 你必须" 慢下来", 而不是" 跟着她" |
| 感到"我应该给她方案" | 你的" 顾问" 模式 | 你的角色是"治疗师", 不是" 顾问" |
| 感到"被她的' 不需要' 推开" | 你的" 被需要" 模式 | 她的" 不需要" 是她的" 3号" 在工作 |
| 感到"我也想做事" | 你的" 行动派" 模式 | 警惕: 你的" 行动" 跟她的" 行动" 可能互相强化 |

### 5.2 督导关键问题

- 这次咨询, 我是" 跟她一起解决问题", 还是" 帮她看见自己"?
- 她的" 成功" 让我舒服吗? " 舒服" 是不是我逃避深度工作的信号?
- 我是否被她拉入" 顾问 - 客户" 的关系, 而不是" 治疗师 - 来访者" 的关系?
- 我是否在" 肯定" 她的成就, 而没在" 看见" 她?

### 5.3 转介信号

- 出现解离/现实感丧失
- 出现进食障碍的临床症状
- 严重的工作狂, 已严重影响健康
- 出现" 成瘾" 行为(购物/工作/性) 作为" 自我调节" 的工具

### 5.4 投射性认同细节(v2.2 新增)

这是 3 号咨询师最需要读的子节。

#### 5.4.1 3 号的投射性认同 3 个阶段

阶段 1:来访者把"评判"投射给咨询师
- 来访者在咨访中感到"咨询师也在评判我"
- 咨访关系越深,这个投射越强
- 这是 3 号无意识的——他不知道自己在投射

阶段 2:咨询师真的开始"被" 拖进投射
- 咨询师真的开始"打分"(这个 3 号做得很好、这个做得不好)
- 咨询师真的开始"挑刺"("你可以做得更好")
- 咨询师真的开始"过度赞美"("你太厉害了!")

这是投射性认同的"关键"——不只是来访者投射,也是咨询师共振于来访者的"我必须做对"。

阶段 3:咨访关系真的被验证
- 来访者看到咨询师挑刺 → "我的判断是对的"
- 来访者看到咨询师过度赞美 → "我的成就是被看见的"
- 内在图被验证

#### 5.4.2 3 号咨询师最常见的 5 个"反投射"错误

| 错误 | 描述 | 后果 |
|------|------|------|
| 打分 | 咨询师开始"评价" 3 号(好/坏) | 验证"咨询师也在评判" |
| 挑刺 | 咨询师开始"建议"、"挑刺" | 验证"咨询师也看'门面'" |
| 过度赞美 | 咨询师开始"过度赞美" | 验证"咨询师也只看见成就" |
| 催促 | 咨询师开始"催促" 3 号做事 | 验证"我不做事 = 被抛弃" |
| 被利用 | 咨询师接受 3 号的"用"(咨询当工具) | 验证"我是工具 = 有价值" |

#### 5.4.3 5 步反投射策略

| 步 | 做什么 | 话术 |
|---|-------|------|
| 1 | 停 | 觉察"我有反移情"(想打分/挑刺/过度赞美) |
| 2 | 命名 | "我有强烈的 [情绪]——这是我的还是你的?" |
| 3 | 不进入 | "我选择 不扮演'挑剔的观众'/'盲目的粉丝'" |
| 4 | 接住 | "你的'成就'/'失败'是OK的——我接住你" |
| 5 | 修补 | "我刚才说了'你可以做得更好'——这是我的'挑刺'——对不起" |

#### 5.4.4 3 号最需要的 3 个"破裂-修复"时刻

时刻 1:咨询师无意中挑刺
- 咨询师说:"我刚才说了'你可以做得更好'——这是我的'挑刺'——对不起。你是你 — 不是你的项目"

时刻 2:咨询师无意中过度赞美
- 咨询师说:"我刚才说了'你太厉害了'——这是我的'过度赞美'——对不起。我应该看见你这个人 — 不只是你的成就"

时刻 3:咨询师无意中催促
- 咨询师说:"我刚才催促你做事——这是我的'催促'——对不起。我应该让你'不做事'也OK"

## 六、自助练习

### 6.1 给来访者的日常练习

练习 1:" 我想要" vs "我应该" 日记(每日)

每天睡前, 写下:
- 今天, 我做 了哪些事? (列出来)
- 这些事, 哪些是" 我应该 做的"? 哪些是" 我想要 做的"?
- 关键: 让她看见" 应该" 和" 想要" 的差距

练习 2:" 不做" 时刻(每日 10 分钟)

每天找一个时刻, 主动 不做任何事:
- 不工作 / 不学习 / 不看手机 / 不" 进步"
- 哪怕只是 10 分钟——坐着, 躺着, 发呆
- 关键: 让她体验" 我不做, 我也存在"

练习 3:" 我是谁" 反思(每日)

每天找一个时刻, 关掉" 角色" , 问自己:
- 如果没有 我的工作 / 成就 / 身份, 我是谁?
- 这个问题的第一反应 是什么?
- 写下来, 不评判

练习 4:" 慢下来" 身体练习(每日)

每天找一个时刻, 做" 慢" 的事:
- 慢慢吃饭 / 慢慢走路 / 慢慢呼吸
- 不是" 高效" 地做, 是" 慢" 地做
- 让她体验" 慢" 不是" 浪费" , 是" 存在"

### 6.2 给伴侣的指南

他的" 做事" 翻译表:

| 他在说什么 | 他实际想说的是 |
|------------|----------------|
| " 我最近很忙" | " 我不敢 停下来, 因为停下来, 我不知道我是谁" |
| " 我在应该 …… " | " 我不知道 我想要什么" |
| " 我已经处理好了" | " 我没 处理好, 但我不知道 怎么开口" |
| " 我们应该 怎样" | " 我害怕 面对我们真的 怎样" |
| " 我没时间感受" | " 我害怕 感受, 因为感受会让我 停下来" |

伴侣可以尝试的回应:
- ❌ "你怎么这么忙" (这激活他的" 我必须忙" 防御)
- ✅ "我不 想让你更忙, 我想 让你慢 下来——你可以 不忙"
- ❌ "你应该多陪陪我" (这是指责, 不是邀请)
- ✅ "你不 用做什么, 我就 想跟你一起 待一会儿——可以吗?"
- ❌ "你从来不表达感受" (这激活他的" 我不能 表达" 防御)
- ✅ "你不 用说很多——如果你能 说一个词, 你今天是什么感觉?"
- ✅ 关键: 不" 否定" 她的成就, 不" 追" 她的情感——给她" 不做" 的空间, 让她自己选择" 慢下来"

## 七、预后与长期维护

### 7.1 预后良好的标志

- 能识别"我在做事而不是在感受"
- 能在"停下来" 的时候, 不恐慌
- 能在"没成就" 的情况下, 接纳自己
- 能在"应该" 和" 想要" 之间, 选择" 想要"
- 能在" 角色" 下面, 看见" 自己"

### 7.2 预后不佳的信号

- 在咨询中"解决问题" 越来越多, 看见自己越来越少
- 转向" 空/麻" (3 → 9 压力方向)
- 出现工作狂升级, 严重影响健康
- 出现" 成瘾" 行为(购物/性/酒精) 作为" 自我调节"
- 反复更换咨询师, 每次都觉得"这个人帮不了我"

### 7.3 长期建议

- 持续做"应该 vs 想要" 日记, 至少 6 个月
- 发展一个" 不做事" 的爱好——散步/发呆/任何让她" 慢" 的事
- 警惕"咨询师 - 来访者" 之间的"顾问 - 客户" 移情
- 每年做 1-2 次"咨询复查"

## 八、临床思考:几个常见疑问

Q:她是不是" 没感情"?

A: 不是。她的" 没感情" 是保护, 不是" 真的没"。她的感情被"做事" 替代了——咨询师的工作, 就是帮她慢慢" 看见" 那些被"做事" 盖住的感受。

Q:她是不是" 不爱我"?

A: 不是。她的" 做事" 是她" 爱" 的方式——但这种" 爱" 太" 远" 了, 伴侣接收不到。她真的 在, 但她" 在" 的方式, 不是伴侣期待的" 在"。

Q:她需要" 放弃成就" 才能被治疗吗?

A: 不是。治疗的目标是让她" 区分" ——做事是"我选择 的" 还是"我必须 的" ; 成就的" 工具" 还在, 但她" 不再被它控制"。

Q:她和回避5号, 在咨询中怎么区分?

A: 看" 退缩" 的方向。
- 5号: 退入内心思考, " 我想想" / " 我分析一下"
- 3号: 退入做事, " 我做做看" / " 我试试"

5号的" 退缩" 是" 不做", 3号的" 退缩" 是" 做得更多"。两者都是" 回避情感", 但3号" 看起来" 很有能量。

Q:她和" 工作狂" 怎么区分?

A: 一般判断:
- 3号回避型: 她的"做事" 背后是" 我必须被爱" 的恐惧; 她的" 角色" 是她真的 信的(她真 觉得" 我就是我做的事" )
- 一般工作狂: 可能是压力应对 / 上瘾, 不一定涉及" 身份" 层面

3号回避型的" 做事" 是她" 身份" 的一部分, 不只是" 行为"。

> 临床箴言:
> 3号回避型来访者最难治的不是她的" 成功", 是她对" 成功" 的执着——" 如果我不成功, 我就什么都不是"。
